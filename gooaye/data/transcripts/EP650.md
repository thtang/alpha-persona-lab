## 瘦瘦針第二針進度

好，那已經進入第二針瘦瘦針了。我覺得現在體感就已經是有感受到藥物的威力了，那只是還是沒有像一些身邊的朋友，或是像我岳母這樣會對食物感覺到反感、或是想吐，那或是噁心一點都沒有。反而呢，對食物是充滿了好奇跟新鮮感，好像回到那種就是國中可能打完球，然後想要吃東西的感覺。每次只要時間一到，食慾就會整個開上來，超級想吃。

可是呢，像之前跟大家分享，打第一針的時候，你大概覺得你的胃口就縮了 10 到 15% 左右，現在大概縮了三成，所以我覺得非常滿意這樣的一個狀態。甚至沒有意外的話，因為我最近看自己的醫師朋友有貼給我說口服都要上市了，如果可以這樣一路吃的話，好像也是一個蠻不錯的選擇。總之我個人是認為說，按照現在的一個進度的話，應該可以非常有效地去控制自己的體重。雖然我老婆都講說這個是作弊，「拜託你有一點骨氣好不好，你就去運動，你就不要那麼愛吃就好」，但是對我們這種人來講就很難，因為我人生的紓壓可能就只剩下打電動跟吃東西，所以如果你不讓我吃東西的話，我很憂鬱。

那像自己身邊的朋友有跟我聊到說，他自己也有去試了瘦瘦針，但他很快地發現說他喝酒就會想吐，然後吃東西的時候就是沒有任何的食慾，所以他就覺得他過了非常憂鬱的幾個禮拜，然後就決定要自行停藥。那同時他也發現說，他自己瘦下去的都是肌肉的部分，脂肪就還好；但是他的同事呢，那可能就瘦了蠻多的脂肪。那所以呢，我也去持續地使用體脂計去追蹤一下說，到底自己掉的是哪一塊。那目前看下來呢，就體重是穩定地往下探，那肌肉量都有維持。

那自己身邊也有一些大哥，那種年紀稍長的，他們也是有在使用這個針劑的，那有跟我提醒說你還是要一直去運動。就這個東西打下去之後，他們身邊有蠻多朋友都會不想要運動，就你整個活動力會下降，那可能跟那個慾望下降有關係。因為像醫師有跟我講說你的性慾會掉下去，我想說性慾掉下去好像也沒差，因為小朋友都生兩個了。

那總之呢，我發現我自己還是有這個運動的動機，就是時間到都還是會想去動一下，或是去上各種有氧的課程。那所以目前算是適應得蠻好，但搞不好到第三針、第四針我的說法也會有所修正。就像我現在在第二針，那我的說法就跟第一針比起來，就是我現在食慾大概就在往下掉了大概 10 到 20 量，那食慾還是非常的強大。所以大概跟大家分享自己的一些人體實驗。

## 中東戰火與市場觀察

那這個禮拜市場呢，就還是蠻急歪的啦，我覺得現在的股票就真的算是很難做的一個狀態，那特別以美股又比台股還難做。所以大家可能會想說，幹已經聽你哀了兩三個月，那但如果說真的在市場裡面有參與的，就會知道說我們這些哀都是有道理的，因為現在的盤面真的是屬於比較難搞的一個狀態。

那同時呢在這個週末，就是在剛要錄節目之前呢，那有看到說在五角大廈這邊他們有一個 Pizza Tracker。那什麼是 Pizza Tracker 呢？就是他們會去追蹤一下附近的 Pizza 店下單的流量狀態，那只要開始發現說很多人在訂 Pizza，那代表說美國可能要有新一波的動作，因為大家要留下來要熬夜，要在辦公室不能離開，所以就會叫 Pizza。那所以呢，搞不好在這個週末又再出一些狀態。

那我們目前看在荷姆茲海峽與伊朗的一個戰鬥之中，就是美國目前是又掉了兩架飛機嘛，一個是 F-15，然後另外一個是一個地面攻擊機。那 F-15 這邊我們就有注意到說，有一個 Pilot 已經被救回來，那另外一個武器官還在搜索的一個狀態。那現在最新的資訊是怎麼樣，我還不太確定，但總之是有這樣的一個發展。

那所以呢，某個程度上我覺得可以視為說，因為以美國來講，他們現在最大的壓力一定是在於說他們戰士、或是說他們裝備的一些減損跟傷害，那這些東西呢只要傳回國內的話，可能都可以鬧到很大。那也開始注意到國內的一些就之前算是 Pro-Trump 的意見領袖，我覺得像是 Joe Rogan，那或像是 Theo Von，在他們的訪問裡面都有提到說就他們是反對這個戰爭。

那但有時候我覺得這種事情很好笑，就是我覺得每個人是這樣，術業有專攻。所以像我們自己可能會去評論一下，可能一下他的一些政治的決策跟我們的股票的關係在哪，可是我不會去評論可能我自己不是特別理解的東西。但我發現就是有很多的網紅，他們之所以後面會走偏掉，或不一定是網紅，他可能也是專家，專家兼網紅，但他們就是撈過界，他們就整天想去評論一些自己不知道的東西，那最後面可能就會貽笑大方，或是說就格局不夠。格局不夠看到的東西就會是可能你只想看到你面前的一個東西的發展，可是你不知道 bigger picture 為什麼我們要這樣子做之類的。

所以我自己現在還蠻買單，就像美國這樣的一個腦袋。有些人會說它是什麼侵略行為或什麼的，但我認為它有點像是，我最近看 Bill Ackman 轉貼一個貼文講說，這有點像是說你在可能日本去偷襲珍珠港之前就先去打日本，類似這樣的一個道理；然後或是在希特勒可能準備要去弄波蘭的時候，你就先去弄他。

那我會覺得這個東西我算是還蠻買單的一個主因是因為，當你看到伊朗遭受美國攻擊之後，然後他的反擊對策就是瘋狂地去轟周邊的盟友，即便這些盟友可能有一些跟他們關係是很好的，他就去轟他。所以你就可以很明白地去了解到，美國的顧慮、或是說周圍一些盟友的顧慮是完全合理的。就如果說他遭受攻擊，他有壓力的時候，他就會去做出這樣子的一個反應，然後他可以無差別地去轟他隔壁的一些國家，即便有些跟他是盟友，或是甚至他會去轟一些可能對大家來說是在生存上必要的一些設施。那這樣是不是如果他今天去持有核武的話，他就是有可能他遭受壓力的時候、或是他今天心情不好的時候，他就會去發動攻擊？

那更別提說在整個中東地區，其實我們會知道說有很多的就是在我們的角度看來是作亂的一些團體，像什麼真主黨啊、或什麼呼希組織之類的，那這些東西其實都是跟伊朗是高度綁定、高度相關的。所以某個程度上我會認為說，就是美國有點像是說，他自己先提早去把這個卡在喉嚨的魚刺給拔掉。

那如果說後續是往一個好的方向發展，就是說他真的去把伊朗的可能做出核武力的這件事情給 delay 個不知道他媽的可能數十年之後的話，那就代表說可能後面會迎來更好的一波那種和平的、怎麼講、就無戰爭然後跟發展的時期，應該是會往更好的方向。那如果說他沒有這樣子做的話，那當然就是代表說，平行時空你會注意到可能油價就不會漲，那就不會有什麼股市的修正或什麼，但可能未來某一天就突然出一個大事之類。

所以有點像是就是大家很常會去討論的，你到底是要先提早享受你的生活，還是你要在後面再去享受你的生活？那它沒有一個絕對的對錯。那只是如果今天是讓一些所有 KOL 或是一些專家跟網紅去做評論，他們可能只會評論到他們面前的東西，就是「你現在就是讓我的價格漲上去，我就通膨就不爽」或什麼的。但如果說你去看更大的一個地平線的話，你就看遠一點的話，你就會知道說其實這個事情本身它某個程度上是化解未來很多的不確定性。所以我自己會認為說，就吞下了痛苦，就是把它扛過去之後應該後面會有更好的一個發展。

那當然不排除就是像我們跟大家聊到的，如果這個油價持續維持在高檔，然後後面又再持續往上開、又再突破新高的話，那真的可能會產生一些可能像是訂單的中止或什麼的，那到時候就是有機會出一個比較大的回檔，那到時候我們可能也會再去修改自己的一些策略。但現階段都還是把它視為是某種程度的，就是大格局上的多頭的回檔來看待。當然如果說後面持續破底，然後跟油價持續破高，那可能想法就會再去做一些轉變，策略也去做一些變化。

所以我們節目的分享呢，其實是比較偏向，因為我們的工作就是在市場裡面去做交易，去找各式各樣的產品來做拼湊跟組合，那所以呢我們比較不喜歡去當一個預言家，就跟大家講說未來一定會怎麼樣，因為未來是含不準的。那像我自己呢，就像說在 2022 的時候，一開始我也相信烏俄戰爭不會打，就很多人其實都這樣相信吧，到後面就打了。

那很多人就會去爭說到底你有沒有猜對烏俄戰爭打不打，但那其實真的不是重點。就像是你有沒有猜對 AI 會在 2023 區域做一個爆發，根本不是重點。你猜對 so what？就如果你猜對是在 2017 就猜對，那你在 2017 就瘋狂去做所謂的 AI 相關的東西，你就不會賺錢嘛。那所以其實猜對並不是重點，重點是說事情發生之後怎麼樣去做處理，那才是一個重點。所以我們的焦點不會放在如何去做預判事件上，而是如何地去在事件發生之後去做處理。

那即便你可能會看起來慢一點或怎麼樣，當然這個也算是一個老生常談。就是你很常會在一些股票討論局看到人家講說，那個標的他為什麼什麼漲了 10、20% 他才講，或是為什麼哪一家券商已經什麼漲了 2、30% 之後他才發報告——這個就是格局的問題。因為漲 2、30%，他後面可能可以漲 200%、漲 500%、甚至漲到 1000% 去，只是他沒有漲之前你怎麼知道他在漲？

所以這個是一個可能會有一點偏向是大家要去思考的一個東西。但我知道其實蠻多剛入門的人會搞不清楚狀況，他都會覺得說東西他就是要買在最低檔。但憑什麼你買最低檔？憑什麼內部的那些不會買得比你更早？就是很多人都想要去——怎麼講——不勞而獲，但事件沒有這麼單純。大概是這樣。

## CPU 題材：Agentic AI 的新主角

那這個禮拜我自己還是持續地去做更多股票的佈局。那在美國這邊我開始把注意力放在 CPU 相關的一些標的。那雖然其實像這樣子的討論大概已經在過去的三到五個月左右有持續地發酵，但我自己還是會去看這個股票的形態做為一個輔助。那特別是當我最近注意到像是 ARM、像是 Intel、像是 AMD，他們算是在美股市場裡面是相對比較硬的東西。那或是說如果你今天轉頭看一下台灣的話，你看一下 3533 嘉澤，它就是跟 CPU Socket 是高度相關的一個標的，那這些東西都算是相對強勢。但台股強勢的東西其實很多，所以看美股比較準。就是在美股我覺得說像光通、然後或像是 CPU，他們現在是走得相對比較好，那所以有可能他們就是帶有一些故事性。

那我們先大概簡單講一下它這個背後故事性的理由跟主因是什麼。那首先大家都是非常明白，在 AI 的訓練時代，就是如果說你有更大的這個 GPU 的衝擊、你有很多的 Rack、然後你有大量的 GPU 晶片，就代表說你可以產出更好的一個訓練效果，這個已經是共識，這已經是完全不用去做爭論了。

但今天我們已經走到 Agentic AI 的一個時代，也就是說過去我們跟 AI 的一個對話會比較像說它是一個 Chatbot，那我就跟你問某個東西，那你就回答我某個問題。那這個在 GPU 的使用方面是無庸置疑的，反正就是誰的 GPU 大可能就可以帶出更好的一個表現。那當然如果說大家最近有在使用比較新的、或者說所謂的 Agentic AI，就是現在大家在 AI 應用上應該注意到說，他們會花很多時間在 reasoning，然後他會有很多 back and forth 的東西、他會有很多 in and out 的東西，東西跑進來跑出去，那接著是不是要換另外一種工具，然後以及呢是不是要去執行某個程式、要去查哪一些資料庫、是不是要去發 API、要去解析一下各種輸出？那這些東西呢其實都算是在 CPU 的工作範圍裡面。

所以你可以把它想像成說，就是 GPU 它算是場上一個非常強的球員，就你叫他去砍分，他就瘋狂地給你砍分；那 CPU 就比較偏向是教練，就在旁邊去做指揮的。那在過去的一段時間就有一點偏向是，我們身為在旁邊看球的人，那我們就是直接，有點類似下棋，我就把這個比較強的球員擺上去，那誰的球員大就可以去無腦砍出更多的一個表現。那可是如果今天讓整個工作流是變得更加地複雜的話，那就需要有一個指揮調派的過程，那這個指揮調派就會使用到大量的一個 CPU。

那如果說我們今天去看一下國外的一些論文跟討論的話，會發現說就像說如果你今天只是去跟一個 Chatbot 對話，或是你要去做訓練的話，那 GPU 它使上的力大概會跑到 80 到 90% 以上，那接下來才會是 CPU 或是其他的元件。那可是如果今天走到 Agentic AI 去使用大量的，就是有很多的 input 跟 output、然後以及可能要去傳換 API、或者是去梳理跟執行程式碼的話，我們會注意到 CPU 它發的力甚至可能會過半，就是會跑到更多的一個用量去。

所以在過去一陣子其實有蠻多人都已經在呼籲說，在接下來我們就要開始去注意 CPU。直到我覺得某一件事情是確立了這個趨勢的發展，就是你在 NVIDIA 最新的展會上面看到說，他直接端出了 CPU 的 Whole Rack Solution，就是整個 Rack 都是幫你塞好塞滿 CPU，同時 Meta 就是用戶之一。所以大家就開始注意到說，在所有的推論或是 Agentic AI 的時代上面，那 CPU 它其實變成是從本來的配角，然後算是搖身一變成是某個程度上的一個主角。

但我覺得說也不用說捧 A 踩 B，因為很多人的論述方式都喜歡，就像說像這個 GPU 很好的時候，它就說 CPU 就這麼垃圾；那 CPU 現在好就說 GPU 不行湊掉。沒有，就是說大家都是同時的發展，那只是在現在這個階段。那未來的階段我們確定，未來可能會再翻轉，AI 時代什麼東西都改變得太快，那現在這個階段 CPU 就是一個重點的項目。

那過去幾天就花了一些時間跟身邊的朋友去討論一些沙盤推演，就稍微地去了解看看有沒有可能在 CPU 這邊也出現一些缺貨漲價的可能性，或是一些意外的發展。那什麼叫做意外的發展呢？照理來說大家都知道說 AI 最缺的是什麼？HBM。然後再來是什麼？eSSD。結果沒想到 cSSD 可能在某段時間它漲幅是超級恐怖的，然後是超越 eSSD。那再來呢，就是大家以為說 HBM 是一個最缺的東西，那它應該是在毛利或是說整個價格組合表現上面最好，結果沒有，這個表現最好其實是 DRAM。

那所以是有可能會有這樣的一個狀態產生，就是當大家可能都把目光放在 server 的 CPU 的時候，那有沒有可能 consumer 的 CPU，就是在大家一面倒的不看好之下，因為大家都知道說像今年的不管是手機啊、筆電或什麼，都已經是開了一個非常爛的 focus，就大家都知道說它出的那個量不會是好的，那會不會反而在這樣的狀況之下突然間給你一個 sexy back 之類的？所以我們就去看了各式各樣的一個可能性。

那我自己一般去找題材做研究的時候，都會是發想於可能自己的某一種想像、或是說我可能看到了一些報導、然後可能看到了一些論文或是一些討論，他們提到了某一個事實，那就用這樣子去做更多的分析。那可是如果要去做確認的話呢，那就會去看一下就是各地的公司派他們家的說法是怎麼樣，因為他們是業界第一線的人。那我覺得這個對於我們股票宅來說是還蠻重要的。

為什麼？因為股票宅很多時候會過度地 sugar coat，就是你很喜歡某個題材，那你去查了一下，發現說他在技術方面是真的很好，或說論述方面是正確，那可是最後面卻發現你可能買進了標的不會漲。為什麼？因為第一個，你可能看的是未來的某一種 RD Roadmap 的東西，就是你看得太早。因為你可能覺得說這是很酷的東西，就類似說大家講說量子電腦量子電腦，可能 5 年前 10 年前就有人跟你講量子電腦，可是直到可能過去的一兩年才成功炒股票成功。然後當然如果說你太早買的話，那一樣是不會賺到錢，那或者是真正的放量可能是在未來的某一天之類，但不會是現在。

所以我們蠻常都會在自己可能去做了一些研究發現說有戲之後，然後我就會去看一下現在可能在企業端這邊大家是怎麼樣去看。首先我有看到 Lisa Su，也就是 AMD 的執行長，她有提到說在 Agentic 的工作流裡面，AI agent 執行大量的任務，它其實都是傳統的 CPU 的任務。同時在 Intel 的 CFO 這邊有看到她講說 misjudge the pace of demand，也就是說他們錯估了 CPU 的一個需求，那 Xeon 6 目前的產能是 absolutely constrained，就是是屌缺的。然後在 NVIDIA 這邊，Jensen 也有提到說未來會有更多獨立的 CPU 的部署。所以你看了一下御三家就可以確定說，OK 那 CPU 的 trend 絕對是正確的，那只是呢它會是怎麼樣的一個受惠的形式，那是由誰會拿下勝利。

那首先如果我們今天來看御三家的話，就會知道說在 X86 方面呢，AMD 其實是慢慢地去吃掉，應該不能說慢慢，其實蠻快地去吃掉 Intel 的一個市占。但兩邊如果說是在這個 CPU 的一個 AI 訓練槽之下，都還是有可能會持續地去做增長。就即便可能 AMD 這邊他吃掉了蠻多的市占，但不代表說 Intel 不會增長，就是說一起增長，那可能就是 AMD 的成長性會稍微地再好一點。那特別是在 Intel 這邊，我有看到它在 NVIDIA 的 DGX Rubin NVL8 的 host CPU 上面呢是有一個這個舞台的啦，就是它是有一個空間跑出來的。那所以會覺得說這兩家應該都是有機會會一起受惠。

那同時呢，大家可能去看到 CPU 好就會去直覺想像說，那最主要在做 GPU 的 NVIDIA 會受到很大的影響。但因為它本身是有，像它最新的這個 Vera Rubin 的平台，它是有 88 核的 ARM 架構，然後搭配自研的 Olympus 核心，所以它本身是有 Vera 的 CPU。那 Vera 的 CPU 看起來也是有獲得客戶的一個採用，所以它本身你某種程度上可以視為說它是一個 CPU 相關的類股。

那只是在我們自己過往的經驗是這樣，就有時候我們在去挑選一些標的的時候，我們會喜歡挑純度高的，也就是它如果在過去曾經某個理由把它拉到很高的一個位置，然後它現在有一個新的故事，那新的故事不一定會推得動它；那反而是過去可能基期很低、就狗都不要的標的，那現在可能會搭配這個新的故事，那也因為它的純度是比較高，所以有機會推得比較高。所以我自己去做一個組合的話，我認為說就假設今天要看 CPU 的話，應該還是會先回到去看 AMD 跟 Intel，那 NVIDIA 可以把它當成是「就你知道這個 CPU 的數字會往上拉，那可能整個模型可以去做一些初步的調整」。

那再來 ARM 可能也是一個在市場上大家可以去注意的一個標的。那首先我們先不去談它的估值，因為這個估值絕對是非常難算的，但同時我們知道 ARM 的伺服器市占率，從大概 2018 年的時候是低於 5%，到現在 2025 年是已經跑到 20 到 25%，甚至在這個 CSP 裡面，也就是 Hyperscaler 裡面，就是佔比跑到 50% 左右。那特別在一些 ASIC 方面也有很多的應用，舉例說像是 AWS 的 Graviton 第五代、或是 Google 的 Axion、或是 Azure 的 Cobalt 200、Ampere 等，他們都是有在使用自研的 ARM CPU。所以 ARM 也算是相關的一個類股之一。

那所以接下來的一個觀察重點就是，以我自己以這個市場操作者的角度來說，就是如果說我持續地看到 Intel 跟 AMD 往上走的話，那可能就是在建倉之後會再去做更多的一個加碼。那估值方面可能不會是我考慮的一個重點，先以題材這方面去做一個布局，那接著再看後面的發展是怎麼樣。那如果說它是，舉例來說它可能突破了一陣子之後，然後開始出現一些跌破拉回的話，就停止加碼，因為可能這個題材就是壓錯，然後或者說它其實沒有像想像中這麼好。

好，那我們目前去看一些可能像是海外的一些研調或說論文，就像說像 Georgia Tech 有出了一個論文，那它就是提到說 CPU 在 Agentic 的工作流裡面，總延遲的比例是跑到了 50% 到 90% 左右，也就是 CPU 已經變成是一個瓶頸。那或是像是 Cornell 大學的研究也有提到說，Agentic AI 的 CPU 佔比是很高的，在傳統 AI 可能是只有到 10% 左右，那可是到 Agentic AI 是有機會跑到 40% 到 50% 左右。那所以應該是可以確立這個趨勢是沒問題的，那只是到底會貢獻多少、會發酵多少，那可能還要看後續產能是怎麼樣去做表現。

那因為我們知道說，現在其實在 AI 的一個很大的瓶頸之一就在台積電這邊，那台積電它的瓶頸之一呢就是在無錫市這裡，所以就算你想要有更多的產能，你也不一定拿得到。會不會可能最後面變成說，是 CPU 跟 GPU 要去搶產能，那搶不搶得到那也不一定，所以可能還有蠻多的變數要去觀察。但是至少自己會開始觀察看看。

那至於要怎麼樣挑這些東西，這就是怎麼講，你研究能力的一個差別啦。就連類似說不知道，其實每個人都是去工作賺錢嘛，簡單來講生活中的每個人都這樣。可是為什麼有些人是郭台銘、有些人是黃仁勳？他們也是工作賺錢，可是有些人是不知道你只能夠去打打零工之類的混口飯吃。為什麼會有這樣的差別？沒有，那就是認知程度的一個差別。

那其實看你的問題就知道說你還在很前面的階段，所以你會覺得你撞壁是很正常的，你不要太容易地認為說什麼你只要隨便研究就可以把大家打爆。每個人——哪有，自己來講我一天十幾個小時，你要拿什麼跟我拼？你當然有可能拼贏我，就你可能天分比我好，我怎麼？但是有機會。只是看你的說法，就是你應該還在很淺的地方，所以你不要去認為說什麼你只要隨便研究一下就可以把人家打爆。你去做幾個週期之後你會慢慢理解，有時候甚至是需要自己給人家不知道撞個灰頭土臉之類的。

像我們之前在做那個瓶蓋股的時候，以前這樣做是有效的會賺錢，後來就沒效了，那為什麼後來會沒效你就會慢慢地理解嘛。有些東西就算你看對，可是市場不喜歡啊。市場喜不喜歡怎麼看？你後來就懂嘛，就看價格變化就知道了。這個東西如果真的大家喜歡的話，那個價格變化一定會有感到讓你發現，就是大家會去罵說「啊炒作啊噴那麼高，一堆人亂買，法人亂買」——那個就是大家喜歡的東西。可是真的發生什麼你敢不敢買？很多人又不敢買，「我怕高」，那這樣去買便宜的東西，那便宜的東西就不會炸，就算他可能真的有受惠，沒有人喜歡還是不會動。

所以最終股票還是回歸到供需，那你所有的研究不管什麼基本面、技術面、籌碼面，那個都只是去想辦法去了解這個供需的變化是怎麼樣。老師跟他傳播一下。
