## 有史以來最 Chill 的農曆新年

準備要開工大吉了，希望大家在新的一年都非常順利。

我自己應該是過了——從高中大學畢業以來，第一次這麼 Chill 的農曆新年。過往節目大家應該都知道，第一我超級討厭放假，第二我非常期待每天的開盤時刻，所以每次放長假都可以感受到我的那種不悅感，甚至偶爾還會抱怨：「怎麼又放假、怎麼又颱風假……」

後來我發現去抱怨放假會被人罵，因為絕大多數人是喜歡放假的，不像我們這種很喜歡上班的人。所以後來就從善如流，不太去批評了。可是心中還是覺得一直放假不是好事。

這一次不知道怎麼回事，整個身體的狀態可能發生了一些變化。小朋友也長大了——三四歲之後可以自理了，不會像過去那樣亂鬧，陪他玩變成很開心的事情。過去陪他玩也開心，只是會累，還是會希望他去上學讓你喘一下。現在基本上可以整天跟他待在一起。

可能是種種理由，搭配某種宇宙信號告訴我要稍微 Chill 一下。這次的部位調節之後，新年過得非常開心。雖然嚴格來說，部位應該要留像過年前那樣的大量倉位跟槓桿——因為看起來新年這段期間盤子還 OK 的（除了美股，美股真的蠻爛的）。統計上也是要留部位的。

過去我把自己放在比較高壓的環境下，這一次就臨機一動選擇降低部位，結果意外發現了人生的美好，重新回到高中大學以後才有的那種放暑假、放寒假的 Chill 氛圍。

## 仙境傳說（RO）回憶殺

除了放假之外，整個假期就是整天在打電動。

玩了一大堆線上遊戲。昨天還在趕末班車搞「暗黑破壞神 2」（Diablo II），因為朋友跟我說有新的職業術士釋出，趕快去玩。前面幾天就是各種手遊加上復古遊戲——仙境傳說（Ragnarok Online）的全系列，全部都抓下來玩了。

在羅馬的時候就開始玩 RO 的一些新版本，現在又回去玩那種最古老的四轉服。玩得非常開心。

同時也發現，長大了真的蠻美好的。大人的世界就是有資本的力量——以前小時候練功練了半死，現在直接花錢去買別人幫你練好的帳號，或是花錢直接買人家弄好的裝備。一個裝備自己衝好，用機率去套可能也要花一兩萬塊，但找人買五六千塊就買到了。你不只是買掉了時間成本，嚴格來說就算自己弄，你花的錢可能更多。所以直接當台幣戰士，蠻開心的。

回到以前那些地圖——去斐揚打月夜花、打虎王，或是以前那種你根本敬而遠之的 Boss，你現在都可以直接一刀衝進去把它車爆，超級爽。這可能是整個寒假最紓壓的時刻。同時也買了一大堆的寵物蛋，現在 MVP 王也可以變成寵物了，以前你的寵物就是路邊的一般小怪，現在可以把這些大王當成寵物，非常紓壓療癒。

## RO 的本質是聊天

重新回到好像小學時候的感覺。我還記得小時候大家都會有外掛，蠻厲害的，不知道小學生怎麼弄到小黑窗。大家都會上去聊天，所以後來發現 RO 其實是一款聊天遊戲——它的本質就是聊天。大家可能坐在普隆德拉的南門在那邊聊天，有空才去打怪。打怪是附加的，刷裝備是附加的，重點是聊天。

當然到現在，大家都長大了，朋友很多也不玩遊戲了，我們都變成一種孤狼玩家。就算你加入公會，也不是我們 20 年前那種公會了——大家時間少，加上有很多娛樂選擇，時間都碎片化了，比較不會賴在上面說「誰上線了，我們一起去打個王」。比較偏向各自玩各自的。

可是我覺得蠻美好的，就是回去以前小時候的記憶裡面，把那些以前打不贏的王全部車過去，擊爽。

然後以前我小學很喜歡的一個對象，他也有在玩仙境傳說。有一次他找很多朋友去他家，可是就沒有找我。我還很不要臉的拿家裡的電話（現在大家家裡應該都沒有電話了），打去他家問說「我可以去你家玩嗎？」他說不可以。我說好。

可是後來我知道他們去他家玩，同時也掛在仙境傳說上面。我就看到他在斐揚主城旁邊，角色就坐在那邊跟別人聊天。以前很流行這樣——開一個帳號上去聊天，同時家裡有同學在，又在上面跟別人聊。

因為我沒辦法去他家，所以我就把自己的角色走到旁邊——幹，超像 Creeper，超奇怪的人。你就在旁邊這樣看他，看一看……算了，這不屬於我，就走掉了。從那一天開始，我也不喜歡那個同學了。

完全想到了這些回憶，把以前那些地圖都走了一次。以前我很喜歡聖誕村的地圖，也重新去走了一次，聖誕村的副本現在一支就可以去單刷。非常好，美好的記憶重新被拿起來了。

## NIKKE、暗黑破壞神 2 與台幣戰士

今天早上剛好跟一個宅男朋友在討論，我們一起考股票，也一起來打 NIKKE。他現在把 NIKKE 的競技場已經練到出神入化，各種搭配超厲害，我一直跟他說你應該要開一個實況——你比檯面上那些做節目的人都還要厲害好幾杯。第一個是你敢花錢，課金是無上限來課的；第二個是競技場已經練到出神入化。有時候你看到一些實況主分享怎麼搭配，結果他可以再提出一個更好的見解。

但他說算了，開這個還要給人家罵。

其實身邊蠻多這種很厲害的人，因為對我們來說，做個遊戲頻道或旅遊美食頻道太簡單了——那就是日常，你每天都在吃、每天都在玩、每天打遊戲都可以無上限課金，只是你要不要去做而已。

他突然感嘆說花太多時間在打遊戲了，可是我反射性的就回答：其實最後發現，張家倫講的那句話是對的——**人生就是不停在打發時間**。打發時間的東西，重點是要找到好玩、有趣、有意義的。就算它當下看起來是「沒有生產力的」，它會留下回憶。

就像我現在回去玩 RO，超感動。以前那些無所事事的夏天午後，坐在家裡打 RO，雖然那時候感覺是浪費時間，可是如果是空白的時間，去做這些殺時間的事情，最後會留下回憶。你就會記得：30 幾歲的這段時間，跟好朋友在打 NIKKE，玩得很高興。如果不是這樣的話，要回想這段時間你可能只記得在搞股票，甚至搞哪支都不一定記得了。

## AI 時代的遊戲與生活

同時有一件很意外的事——這次透過 AI 的幫忙，玩遊戲時包含自動喝水、各種輔助工具，現在都可以直接 Vibe Coding 把它搞出來。雖然要花一些時間讓 AI 搞清楚你的需求，但過去是不可能自己搞一個外掛的，現在竟然可以自己搞，非常感謝這個時代。

還有一件很好笑的事：跟兒子在玩 Switch 2 的寶可夢的時候，買了一個精靈球造型的控制器，但我們不知道怎麼在 Menu 裡面切換寶可夢——它只有兩個按鍵，有時候可以進去有時候不行，好像很隨機。

結果隨便跟 Gemini 形容說「我有這個精靈球控制器，想要進去換寶可夢，不知道按哪個鍵」，拍照給它看，它就教我怎麼弄，問題就解決了。所以我們真的活在一個非常好的時代。

## 開工展望：台股與美股

接下來進入今天節目的話題。

以最後的收盤時間來看，富台指換算成台指期，基本上大概在 10 點以內的價差。如果在禮拜一的國際盤——早盤的 NASDAQ 或日股韓股——沒有什麼其他變化的話，差不多會開在一樣的地方，No Significant Change。

如果看美股的話就是不一樣的感受了。老實講，美股應該要減碼更多，心中的想法是這樣，只是因為美股都是比較長期持有的標的，就稍微調一點點而已。比較可惜的是美股最近不知道發生什麼事情，整個狀況比較差。

在新的一年，我覺得大家用放假前的態度去面對就好，不一定要立刻把所有部位都補回來。指數差不多停在類似的地方，只是有很多資金——像槓桿型、偏交易面的，以及那些場外金主融資的資金——他們在放假前會撤出，之後會補部位。不一定補很快，但就是會補。

所以如果國際盤沒出什麼大事，台股可能會有一兩個禮拜買盤持續進來，因為過去賣東西的人會補回來。當然如果國際盤有出事，那又是另一個範疇了。

## 伊朗戰爭風險

最近在美國市場發生蠻多事情。第一個比較大的，我個人覺得是對伊朗的戰爭可能要發生。北約以及波蘭的領導人都已經提到，希望大家盡快把僑民撤出來——還在裡面的趕快閃，因為之後可能就沒機會了。

當大家都給出這麼明確的訊號——這已經不是暗示了，就是明示叫你趕快出來——區域衝突的機率就明顯上升了。

戰爭這種東西很難說。以前我也覺得，當大家已經知道有戰爭要發生的時候可能就算是已反應，因為前面早就各種布局了。但 2022 年的俄烏戰爭之後，發現好像不是這回事。市場真的沒有所謂的絕對——有時候反應、有時候不反應、有時候延遲反應、有時候提早反應、有時候過度反應、有時候反應太少。

像 2022 年的俄烏戰爭，那時候大家也覺得就是一個區域戰爭，現在看確實是。你看指數全部跑到新高區，可是那時候就跌了一年。到底是因為戰爭，還是供應鏈去化，到現在都沒有定論。

所以我現在也不會用很輕率的態度說「伊朗不就是世界某個角落」。我覺得還是有可能會有事情，自己會稍微警覺一點。

具體怎麼做呢？我不習慣去把標的大幅賣掉或大砍，只能說假設要買東西會買慢一點，要賣東西會稍微賣多一點。比較細微的調整，讓自己心理調整到：如果後面發生事情不會恐慌想把東西全賣掉，如果突然間突破也不會 FOMO 想馬上追回來。

## IEEPA 關稅被最高法院否決

今天佔據最多新聞版面的是：用 IEEPA 去課關稅，被最高法院給否決掉了。

實際上我覺得這件事不算是一個很大的新聞，算是可以預期的。只是它還是在報章雜誌上獲得很大篇幅。昨天市場的反應是把它當利多解讀——看到東西有在上。但我跟身邊一些朋友討論之後，比較偏向認為這算是利空。

當然我們不會說一兩天的市場反應就代表什麼，很多時候拿短期反應去講故事，高機率會被打臉。重點是做好準備、做好部位調節。

我自己認為這次事件是把不確定性又墊得更高了——偏 Neutral 到負面，而不是偏 Positive。外面有些討論我覺得想太多：什麼川普要怎麼退關稅、企業怎麼提告美國政府。我自己有 98% 到 99% 的信心——這個關稅是絕對不會退的。

如果他現在退下去，第一個他後面不用選了。搞了一齣大戲搞到天翻地覆，然後跟大家說「不好意思，失誤，從頭來」——這個機率極低。他課下去就是課下去了。

接下來他要做的事情很清楚：這個 IEEPA 被擋住的話，他有太多其他法源依據可以用——Section 232、Section 301，還有國家安全的 Section 122，先給一個 10% Universal 關稅之類的。本來用一個大架構、簡單方式課大家，被最高法院擋掉之後，他可能改用比較零碎碎片化的策略，這邊弄一點那邊弄一點，反正該課的還是會課。

而且一些本來跟川普就站對立面的國家——加拿大、歐盟——一定會藉這次機會訴諸民意，說最高法院都這樣判了所以應該更強硬對抗川普。你覺得川普不會再揮動下一次重拳嗎？

所以我個人認為這次事件添加了更多不確定性，而市場是不喜歡不確定性的。禮拜五美股的反彈，用短線來看我覺得怪怪的——如果是跌的話我心中反而比較舒服。有時候看到漲我們不一定會很高興，即便我們是多軍。看不懂的市場，反而更害怕。

## Anthropic Claude Code Security 與資安股

今天早上市場又出了一次針對資安股的圖殺。很好笑，我上禮拜才跟大家分享說覺得資安股應該可以看，然後 Anthropic 推出了一個 Claude Code Security 的服務之後，資安股又全部被打下去。

結論是：又再一次的，過去大家不相信 AI，現在大家變成過度相信 AI。過去不相信的時候我不停地在 Preach，覺得 AI 很厲害；現在大家要尬吹各種 AI 取代一切的時候，以我的個性，還是要去找一些可行性。因為很多時候所謂的「看圖說故事」，你當下看起來是對的——廢話，你看圖說故事當然對。台灣一堆媒體人都在做這樣的事情，大漲就出來寫個漲的故事。做這種事是降低自己的智商，你應該持續維持正確的判斷。

這次的 Claude Code Security 嚴格來說是某種 SAST（靜態應用程式安全測試）工具——你把程式碼寫好之後，裡面可能有些漏洞，AI 幫你弄出來的東西或你自己寫的東西都可能有漏洞，這些漏洞可以在一開始就補好。只是過往用人力去掃，不一定看得到。

但這種靜態掃描工具，在目前美股上市的資安公司裡面，只是其中一部分的業務。它並不是現在主流追求的東西。所以某個程度上，Anthropic 是發表了一個「我可以讓你的 Code 更完善、Vibe Coding 時出現的漏洞我可以幫你補」，然後大家直接跳躍解讀成資安公司都去死——但這跟資安公司努力的方向不一樣。

像 Cloudflare，重點是擋住暴力流量以及在 API 取用上受惠，跟 SAST 的故事不衝突。CrowdStrike 在 Google 或 Microsoft 機房裡面的伺服器掛一個 App，佔極小空間就做全面防護，防的是有人要快速突破時的即時封鎖和阻擋——也跟靜態掃描有一定程度的差距。

當然不排除市場情緒在反應「它慢慢又往前走一步了，會不會最後走到最專業的項目去搶市場」。當這種事情發生的時候，可能產生估值壓縮。但我不會特別悲觀，因為很多新工具出來的時候，人們常常低估新工具加入後導致的 TAM 市場變化。

## 軟體的 API 化趨勢

跟前面在對軟體做估值調整是差不多的道理。

老黃都再一次在訪問裡跟大家說了：他認為外面講「軟體要死掉」是很奇怪的說法。你不會為了使用某個東西就重新發明輪子，那是沒有意義的事情，花費的成本是巨大的。

比較常看到的、實際上在發生的狀態是：所有應用軟體慢慢走向 API 化。也就是說 User Interface（使用者介面）不再是最重要的——現在是讓機器人、讓 AI 直接使用這些軟體，所以只要串好接好就可以。

過去可能 A、B、C 三個軟體背後功能差不多，但 A 跟 B 介面比較好。類似我們之前舉例的下單軟體——大家覺得 Firstrade 介面好、Robinhood 介面好，IB 的介面沒那麼好。過去介面好的就是大家的選項。但如果之後是 API 化的時代，介面就不重要了，因為是讓機器人去看的，能執行更好 Result 才是最重要的指標。

我覺得這樣的發展對整體人類更好。過去可能買一堆 SaaS，不一定都有用到，就是佔空間花錢而已。現在都是用 Result 做計費，不會重新定義「軟體消失」——你還是需要 Application 層面。只是在 Agent Layer 這邊可能會變成最有溢價權的東西，因為它可以決定要支用哪個軟體的 API。最後一定還是會有贏家 API 跑出來，只是在這段劇烈變化期間，市場會有這樣的震盪。
