好，那特斯拉正式公布了他的 Model 3 新改款，叫做 Highland。這個 Highland 就是我之前有跟大家分享說，假設你要買 Model 3 的話，可以等看看後面有一個改款。我當時的說法是說，這是一款在硬體上降規的車子，只是在軟體上面，你可能會感受到一些升規的感覺。可是實際上這台車它是一個 cost down 的結果，對特斯拉來講，它的成本是做得更漂亮，所以在這台車的毛利會很不錯。

那當時有給大家一些我的預估數據跟預判，我們自己考察之後的一些猜想，大多數的猜想都成真，但是最關鍵的價格猜想全錯，全錯。我們身邊在看特斯拉的沒有人猜到，它的價格是遠高於我們的預期。我們本來預估說，高階的車款會賣到 30 幾萬人民幣，那沒有辦法，因為它的設備什麼是真的比較好。可是它的一般版本應該是要壓在 25 萬以下，甚至是趨近於 20 萬人民幣。為什麼？因為現在北上廣深的一個補貼是 25 萬人民幣，它就會去補貼數萬元不等的一個電動車的加碼補助。那這個補助對電動車車廠來說非常重要，因為它會讓這些購買者更有動機去買車子，廢話，有補貼嘛。甚至在中國這邊，很多人會去到有補貼的城市買一台車之後，再開回去自己的戶籍地。

所以這個補貼是一個很重要的事情，我們有一點想不通的是，為什麼它的價格會比它本來的 Model 3 貴了 12%，那貴 12% 還不打緊，還是超過 25 萬的價格，來到 25 萬 9 千。這個是我們很意外的一件事情，就是為什麼它會這樣子搞。我們趕快小組會討論一下，交叉比對之後，我們認為應該是因為 Model 3 的庫存問題還沒有完全解決，所以它會等到舊款 Model 3 的庫存出掉之後，之後可能就會開始去做降價。所以假設是我自己要去買這一台 Highland 的話，我會選擇等待，因為它之後應該會很快降價，我認為會降價。

為什麼這樣說？因為它那個成本結構漂亮太多了。我們本來估計的毛利率 21、22%，高於整體公司平均，那個是在講說 20 萬出頭人民幣的毛利是這樣。它現在開到 25 萬，毛利超高、超級高，所以它那個毛利護城河又拉得更快，它更有餘裕去降價。但它也不一定會降價，好像蘋果的手機越賣越貴，它沒有跟你降價的。只是特斯拉目前的策略，按照馬斯克的說法，就是它必須要以拼量為主，它還是希望可以趕快達到它 2000 萬年出貨量的一個目標，所以它確實是有可能會繼續用降價的方式去排擠掉其他的車廠。

但這樣其實是一個雙面刃，以車主來講，現在你買特斯拉有時候就覺得說「幹嘛要這麼早買，晚一點買又便宜更多」。可是就是回到「早買早享受，晚買享折扣」，晚買有時候那個折扣真的蠻大的。在台灣這邊，Model S 跟 Model X 在同一天就直接降價，Model S 的入門款 300 萬有找就買得到。300 萬有找買得到這種馬力的車子很屌，那已經是類似超跑了。那 Model X 它是直接降了大概 30 幾萬左右，那新的車色是不用錢，所以等於基本上就是直接送你差不多快 40 萬。所以你想你可能早買一段時間就直接貴 40 萬，這其實還好啦，它之前有段時間是直接降了快兩三百萬，那最扯。那時候是真的有車主跑去拉布條，因為真的太不爽了，那個降價還一次把它降五成，太誇張了。

那其實現在特斯拉是位在一個持續降價的趨勢上面，它持續把它車子的價格降低，這其實對整體的人類文明我覺得是一個好事情，對它達成願景也是好事情。至於股東的話，可能我們就要去盤一下說這件事情到底是好事還是壞事。我們當然希望最好的狀況是，你去降價排擠別人的同時，你的成本也是持續維持很好的控管，所以讓你的毛利可能可以鎖在一個比較固定的區間，那同時你的市場又擴大。但目前這幾個條件並沒有同時達成，所以對我們來說，特斯拉我們不敢說是有隱憂，不過就是有一些需要值得觀察的地方。

像這一次大家對他的預期，就認為說這個價格應該要壓在補貼的水線之下，沒有達到，這個是我們很意外的。但其實我們在看特斯拉的供應鏈，老實講很多時候都會 miss。因為特斯拉這家公司，我們推估出來的一個可能性是，很多的決定應該是馬斯克自己決定。他不像很多企業是，我今天決定要做某件事情，我在高管之間都先有一個 road map，大家大概知道怎麼樣做，那有一定的 SOP。那特斯拉這邊就是，你很明顯的感受到，講好聽叫彈性，講難聽一點就是有點雜亂。你可能上週漲價，然後下週又突然降價，又再漲價，又再漲價，就是他那個價格跳動得超快，跟我們打期貨一樣，所以就很誇張。這個決定可能就是高度集中在馬老闆，不然就是可能馬老闆周邊的一群人。所以變成，這個結構可能是比較扁平一點，他們可以更靈活，可是同時他們的一些出招真的會讓你看不懂。

但這次這個價格坦白講，就是真的讓我們很多人跌破眼鏡，你怎麼可能會開這個價格？那往好處想就是說，假設他把本來的 Model 3 庫存都清掉，他還維持這個價格的話，然後這台車又賣得好的話，那個毛利會很可怕，這是真的。所以我們就是往好處想是這樣子，可是往壞處想就是，你怎麼可能不去拿補貼？所以我還是覺得，在機率上他會降價，而且會降很快，他只要把 Model 3 的舊庫存排擠掉，他應該會很快就降價，不然有點說不通，因為那個補貼就放在那邊，為什麼你不去拿？這個是我比較想不透的地方。

那在車子硬體的部分，整個車構件那些，我們就不去細拆，也比較複雜一點，反正就跟你講說他就是 cost down，他的安全性什麼都還是一樣，可是他就是用材料的調整去達到 cost down 這個結果。但是在軟體上讓大家非常有感，所以我稍微看了一下各大論壇，基本上大家對他都是好評。所以代表說，你要去買通消費者的話，你就是把軟體好好弄、內裝好好弄，大家會喜歡弄一個什麼氣氛燈、弄一個車頂的隱藏式冷氣，以及後面送你一個螢幕，大家看到後面有一個面板，幹超開心，什麼通風座椅超開心。所以你那個什麼硬體去降規，那不重要，反正一般人真的也看不到。老實講，你沒有去了解供應鏈、沒有去了解他生產環節的，你也不會知道，因為反正安全性都是一樣的。所以我用的密度弱一點，其實也沒差，一般人也感覺不出來。

不過就是在我們的考察上，我們會認為說他降低成本，那我們就會去研究說，那他降低成本之後，他會找誰來做，誰是供應鏈，那這個量會不會因為這樣放到更大。所以這個是我們會去研究的東西，我才會跟大家提到說「降規」這件事情，因為有一些人就問我說，看起來是升規。我們講「降規」是講硬體的環節，然後在車內的軟體、螢幕，你看到的是提升。所以其實在很多的不管是電子科技業還是車廠也是這樣子，他可能拔配備，只要拔在你看不到的地方，你就不會覺得是什麼大事情，反正他整體讓你感受到車子比之前更屌，那就好了。

所以像什麼電池的續航提升，那其實也不算是把規格拉高，因為電池本來就會越來越好，續航本來就會越來越好。只是就在成本的控管，我們看到他把它壓低，所以我們真的是比較難想像，為什麼那個價格反向的往上走。但還是傾向，好，先蓋一個章，就是說我覺得他是要把庫存清掉之後，這台車就會很快的降價，而且降的速度應該會蠻明顯的。

那這個時間點我們稍微估了一下，除了可能假設他又是不照牌理出牌，在半年內隨便亂搞之外的，不然比較高機率是等到，因為明年 Model Y 也會有一個 Highland 的改版，目前這個機率是蠻高的。那當 Model Y 推出的時候，因為在中國這邊，他們年底就有謠傳說會在一些大城市去試點，要讓他可以弄 FSD 跟美國一樣。所以呢，Model Y 他在明年推出的時候，因為已經有些城市假設都順利，他已經有在弄 FSD 了，所以他就有這個誘因去提升到 HW 4.0。其實這一次為什麼沒有把 Model 3 的 Highland 版本變成是 4.0 的，有個說法就是說，反正現在你還沒有辦法去用到 FSD，所以我去做這個晶片的提升，你感覺不太出來，可是對我來說成本就直接拉高了。

所以他等到明年，可能 Model Y 就是搭載 4.0 的，然後可能之後新出來的 Model 3 也都會是搭載 4.0，但是應該會跟著 Model Y 的改版推出一起推出。屆時呢，當 Model Y 推出之後，可能這個 Model 3 Highland，就是新的改版就會降價。這個是一個比較容易想像的劇本，就是我們覺得是有可能會這樣子發生。

所以我覺得對於大多數的消費者或是投資人來講，其實你沒有要做供應鏈，你不用拆這麼細，你不用去賭說什麼時候會有規格配備的調整，因為對你來說不重要。你如果是特斯拉的投資人，你要專注的是，這台車到底大家覺得香不香？這台車香、大家會買單，你心中放一個底，你知道說其實有在 cost down，毛利是更漂亮的，這樣就好了，這個就是最重要的東西。所以我覺整體來看這個車子，我在網路上逛了一圈，我發現誘因是蠻高的。只是一樣，那個補貼那個我百思不解，它壓在補貼價格之下，應該那個量馬上就刷出來了，為什麼要這樣子？我還是要再觀察一下，但我猜是要去庫存。

那我們可以觀察一下其他地方，可能是歐洲的比利時、德國，因為他們也上架了，然後那個價格也是比舊款的車子稍微貴一點點。如果說在這邊的買單狀況是很不錯的話，就是那些「早買早享受」的衝進去，看起來是不錯的話，那我們應該就可以很快的建構出所有對他的銷售預期的模型。我相信會是一個不錯的狀況，就以目前市場的接受度來說，那其實本來在嘴、在噴的，看起來是蠻喜歡的。但其實我們都知道說，很多時候是萬人響應、一人到場，如果後面實際上銷量沒有出來的話，那當然是另外一回事，可是目前看起來是一個還蠻不錯的起點。

好，那最近市場上當然就在繼續密集要開財報。昨天有一個真的是讓我很意外的，就是戴爾（Dell）的財報，他算是釋出了比較樂觀的信號。可是我們都知道說 PC/NB 市場，已經講過好幾次他在復甦，但其實真的是距離復甦到可能本來很好的狀況，還有一大段距離，但他股價他媽直接噴新高，昨天直接噴了 20 幾趴，超屌。所以我看到他噴 20 幾趴，我跟幾個朋友剛好在談心，我就講到說，你知道未來的生活一定只會越來越困難，那相對剝奪感只會越來越高。我想說，像這種幾十年、有沒有到幾十年，可能十幾年一遇的那種循環低點，就真的很爛很爛，那個營收、獲利每家都開得比狗屎還爛。這樣的一個背景之下，很多公司在創新高。

當然你現在講說他可能是在炒作什麼的，那如果說他真的就沒跌下來怎麼辦？如果說大家真的是 look beyond 怎麼辦？如果大家真的就是直接忽略現在的數字，我們都知道很差了，那就不要再講了，可是之後會復甦，所以我們就進去堆，那估值假設就撐到這邊。就類似說什麼，有時候去看一些歷史，我會覺得蠻有趣的。像之前去夜宿忠孝東路的無殼蝸牛族，那些人如果買房子，現在賺更多了。如果說沒有買的話，就覺得會崩，這個房價這麼貴，一定要打房，那到現在就很意外，就是貴還可以更貴。你當時以為那樣很貴，現在貴三倍。所以股價我覺得有時候是這樣子，遇到股災可能早一點進市場的時候，那種早期會覺得很害怕，可是到後來越看越懂，就覺得說，幹，其實每次股災都要把握，除非這次是人類毀滅。可是如果是人類毀滅，老實講什麼都不重要。所以你每次都要期待說，這不是人類毀滅，所以當很多人在叫大拍賣的時候，那可能真的都是機會。

他講說什麼叫做會做人，我覺得會做人就是，他以前當兵的班長講的：「各位！禮多人不怪，你各位！」他會提到說，反正你就是看到長官就敬禮，你該怎麼樣做，你絕對是對的，這個禮數上是怎麼樣，你就做就對了，你不要覺得說會不會很尷尬或什麼的，反正你多一點禮數不會有問題。那個班長講的到現在我都還記得。所以我覺得會做人的人就是，在吃飯的時候會稍微服務一下大家，在發現人家面有難色的時候會出來打個圓場，那當發現可能今天太太不高興的時候，那你就會趕快 shut the fuck up，趕快去洗碗。如果說要把它總結的話，我覺得就是「會讀空氣」，會讀空氣的人就是會做人的人。

那當然可能還有一些次要的條件，就是會讀空氣可能也要有執行力。像我自己我覺得我是一個很敏感的人，我還蠻會讀空氣的，但為什麼我不太會做人？是因為，可以講說就是因為 I don't give a shit。就是我知道現在有人覺得很尷尬，但是我沒差，因為我知道只要在一個環境裡面，我不尷尬就是你尷尬。所以我跟人家對視或做什麼的，那不是說我想要人家不舒服或什麼，是我知道我應該有更好的做法，可是我也懶得，我就不想要去做嘗試了。

所以我覺得就是說，你要會讀空氣之外，那你還要有這個意願去嘗試。那只是這個意願隨著你的年紀增加，會直線下滑。我認為，為什麼這樣說呢？就是說很多的長輩，其實他不是沒有禮貌，他是懶得演了。他知道禮數是什麼，但是他就是不想要做。他知道怎麼樣叫做會做人，可是他就是不爽這樣子做。他就是覺得他媽的，我已經年輕的時候花了一輩子的時間在逢迎別人，我逢迎得很好，我把每個人的覽蛋都捧得很舒服，但我他媽現在就是不想這樣做。所以除了會讀空氣之外，我認為要會執行，湊在一起才可以變成是一個所謂「會做人的人」。

那有些人他不用會做人，我覺得大多數人會做人都是加分，只有些人他天生就是不用這麼會做人。只要人長得帥、長得漂亮、外型是好的，或是說人脈很強大的人，因為大家要來找他捧懶趴，大家需要他人脈。那或者是財富自由的人，很有錢的人，因為他不需要看任何人的臉色。所以我覺得只要是你達到了心目中可能想到的某個位階之後，你就越來越不會做人。當然也有人是他可能是位高權重，那還是會做人，有這樣的人。有些大哥出去吃飯真的覺得欽佩，這麼厲害還這麼謙虛，這樣的人是也存在，還很客氣。不過我覺得用大數據來說，雖然我沒有真正的數據，但我感受，我們感受也是一種數據的統整，我認為會做人是在你年輕的時候對你很有幫助，然後到老的時候就還好，那個影響可能就有限。因為大家都可能會回歸到自己的家庭，不像年輕的時候可能會很著重在社會上需要去攀爬，所以你要去爬這個 corporate ladder，你可能就需要有會做人的這個特質，那得到後來你可能就慢慢的不需要。所以當你還停留在討論說會不會做人的時候，就是你可能比較傾向於有求於人的時候。因為當你今天都不用有求於人的時候，你不會做人也不重要，就算人家跟你說你不會做人，你也不在意，是這樣。

那再講說，加盟或是買貨車，追求當老闆，是不是就是高級打工族？不知道，我覺得這個沒有什麼好講。就是說每個人其實都是打工仔，那你要硬稱自己是老闆，可以啊。你開計程車，你就是老闆，對吧？你自己個自營，那個就是你的生財器具，你就是老闆，你可以選擇要載誰、不要載誰。可是當今天一個乘客上車，他叫你左轉，你就是左轉，你就不是老闆。我在家，我在那邊弄股票，我就說我很獨立，對吧，其實我不需要去聽任何人的話，我做我自己的東西。媽的，老婆叫我去遛狗，我就是去遛狗，他就是我老闆。

所以其實就是每個人都有自己的老闆。當然我講這個可能是比較歡樂一點，但我們講一些傳統上看起來沒有老闆的人好了。我們講說一些大公司的創辦人兼 CEO，他是最大的老闆，可是他還是在聽命於他的董事會、股東會，那股東講什麼、叫他做什麼，他還是要做，對吧？他不給他媽兩手一攤，他不是全職。好，他就算他是全職的，就像是馬斯克，他買下這個 X，當然他有一些股東，只是我說因為是他收購下市，我們先假設說他就是 X 的老闆，就是以前的 Twitter。那他是不是就沒有老闆？還是有啊。今天這些消費者不爽，不要用他的東西，老實講他就是下去。所以對他來講，消費者就是他的衣食父母，就是他的老闆，就是他的 boss。

所以我是覺得，我們生存在世，每個人都是在為某件事情或是某個人打工，沒有一個人可能是萬人之上，然後沒有在一人之下，你都有一個「一人」，至少都有一個「一人」在上面。甚至那個「一人」可能是你的狗、或是你的兒子，你就是在養他，你就是在逢迎他，你是為了他在工作，你要去考慮他的感受。所以其實每個人都是打工仔，就是不用去很糾結說什麼「老闆」、「做自己的老闆」，那個都是話術，那個要騙你去加盟、去搞你的。實際上就是不要太糾結這個東西。

但我知道很多人就是很在意那種名目的東西。你慢慢去看他臉書，一整排頭銜，很多人家一整排頭銜，很屌，他可能是真的很屌，可是他就是喜歡放一堆頭銜，他可能就是很在意他有一個名，他覺得聽起來很爽。可是有些人就是空白，像我自己就是空白，不是說我比較帥，就是我真的完全不在意，我真的完全不在意你認為我是誰、我該怎麼樣被定義，我完全不在意。所以就是可能每個人對自己的生涯規劃，我認為是這樣子。只是你要說誰是真正的老闆，沒有這樣的東西。那上市公司老闆也是要出來被股東電。所以，當然你可以講到很誇張，就是說人民就是頭家，所以消費者就是老大，也可以這樣說，可是也不是這樣說，因為這個老闆他不爽說他可以隨時抱著一堆錢退休，所以他還是在一個更爽的位置。只是不用去糾結領導地位，到底誰是領導，我覺得那個不是很重要的一個事情。

下面一位，北基雄 Killer：「第五次測試，我先吹捧。有此節目，沒有什麼想問的問題，只是想說，感謝癌大分享說的內容，我相信未來的世界，癌大的 Podcast 還是會被世人所傳唱。最後想說的是，誰來救一下 Disney 好嗎？」

Disney，我就沒有看他了。那時候疫情的時候，還記得那種節目超早期的時候，Disney 一崩下去，我還有進去抄底一下，因為那很直覺，就是那個實體的園區都關掉，可是我們知道他一定不會倒掉，所以就是進去玩過他。現在已經沒有玩他，也不知道後來的發展，只是他那個老闆有回鍋，老闆的公關能力真的很強，在外面都洗得很像說他自己就是一個救世主，但其實很多問題都他自己創造的。後續我們可以再找時間來聊一下 Disney，還蠻好玩的。
