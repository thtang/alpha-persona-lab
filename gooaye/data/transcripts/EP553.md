那經過快要一個月的連續堆肥之後呢，就終於要登出月子中心了。那對我來講是有一點傷感啦，因為在月子中心裡面就是整天飯來張口，什麼事都不用做。那我老婆又不吃月子餐嘛，所以月子餐全部都是我在吃。我們除了一天三餐的月子餐之外，中間餐跟餐之間還會有像是雞湯，或是一些小點心什麼的，那基本上一天都有兩餐的雞湯保底。

然後再來就是，因為那邊的督導對我非常好，整天就會送雞湯給我，也是我們地方聽眾。應該還是氣憤仔啦，反正我不知道怎麼會有這麼多氣憤仔在醫療產業相關的，會持續收聽我們的節目，他們可能覺得還蠻紓壓之類的。然後他就會覺得說，我可能變瘦了還是什麼的，然後就要再額外塞雞湯。所以我大概一天是保底吃五到六餐吧。

但是其實我蠻意外的是，每天吃，然後外加那個步數——只要我待在月子中心裡面，當然中間還是會跑出來運動，或是跑一些文件、自己工作需要的東西，可是呢，只要我待在月子中心裡面，我一日的步數大概就是兩百步而已。就是說你根本就是廢物，你只有站起來上廁所而已。除了上廁所以外的時間呢，就是坐在那邊。我直接帶 PS5 進去打，那就是瘋狂打電動跟吃東西。

很意外的竟然沒有因為這樣子變胖，可能是因為他們的東西比較乾淨。我自己平常在家裡會吃一些可能更高熱量的庶民美食，一些像是什麼炭烤啊，或是麻辣鍋之類的東西，這個是我愛吃的。所以上一次去做身體檢查，最後面發現沒有半顆息肉，這是我嚇到，就是我把它當成是老天給我的一個饋贈跟禮物。因為太扯了，我這樣的飲食習慣，然後整天他媽的坐在椅子上基本上都不動的，竟然在大腸方面沒有找到什麼問題。所以好好珍惜這一切，後面會更加照顧自己。反正當我發現在月子中心沒有變胖的時候是超級開心。

所以這一次的陪產之旅呢，我個人是給了一個滿分。就是有很多的問題、有很多的狀況，但最後面看起來都是有驚無險。那我們還是持續的保持謙卑謙遜，就是不要太早去插很多 flag，也不要太早做定論，還是會持續的去追蹤。但是非常感謝，至今的體驗都是蠻好的，各種化險為夷的感覺。

那也非常感恩，就是有這個節目的存在。所以像在睦生這邊的什麼小兒科醫師、超音波的醫師什麼的，幾乎都是地方聽眾，我也嚇到怎麼這麼多。那他們可能是因為聽了這個節目之後，就跟我變得蠻麻吉，即便我們在現實生活中沒有見過面，可是聽久了之後就麻吉了。看到這些兄弟一下是什麼狀況，直接跟你講。所以因為我這一層關係，就還好平常可能沒有造太多口業，雖然有時候講話可能蠻機歪的，但至少他們可能是還算喜歡我的，所以跟我講東西的時候也都是很直白，給出超讚的建議。所以真的是充滿太多的感激，還有各方的護理師照顧我們的小孩，整天幫他拍照什麼的，這個體驗都是超好啦。

那其實說到月子中心，最近剛好跟老婆在討論這件事情。我就跟她講說，我仔細算一算，我發現月子中心並沒有很貴。當然我們講的不是說以整體總價來說這個錢不貴，不是這樣，就是說假設你是要把每個環節都算進去的話，它不貴。特別是假設你是有一些狀況的話，其實算起來是還 OK 的。

那其實像我老婆最早最早，她根本就不知道什麼叫月子中心，因為她是國外來的。所以當我剛剛講這個東西的時候，她還先吐槽說「是怎樣，這邊女生是比較弱是嗎？就為什麼會需要住月子中心這種東西？」可是等到她第一次住進去的時候，她就說她下一次也要住月子中心。就只能夠說，這個東西可能在海外沒有這麼普及，那最近是開始越來越多，因為像美國那邊就是有很多跑去那邊生小孩的，所以他們就有坐月子，那這個東西也開始會普及到——就是有一些當地人可能也會去採用。所以像我老婆一開始她就不知道這種東西。然後她說她媽媽就是在醫院生完三天就回家。那像他們可能隔壁鄰居或是一些朋友的媽媽，假設是有一點條件的，就會去請一個類似月嫂這種，然後在你家裡，他們也沒有什麼包辦或什麼，就是幫忙你做一點事情，因為你可能行動比較不方便、有一些傷口或什麼的。所以她本來的認知是這樣子，然後就是自己去住了之後就一試成主顧。

我們那天就是很認真在算說，因為像我老婆就是在母乳的供給方面是還蠻足量的，小朋友都可以吃很飽，但是有時候還是會遇到像是什麼卡住的問題。然後我們就是很認真在算說，假設我們回林口，我們要怎麼樣去維持我們現在在這邊的一個生活條件，就在月子中心可以受到的一些服務，要怎麼樣在林口可以繼續去維持。然後就算了一下，發現其實不簡單。

首先先講在台北，你要去租一個大概三到四坪左右的套房，然後是有兩張床、有電視什麼的，差不多以下的房價來講，然後外加在台北市中心的地點，我在想就大概是要五到六千塊左右，那就是有這樣子的一個房間。這只是房間。然後再加奶子專家——我不知道怎麼講，泌乳大師也不太對，奶子專家、泌乳大師不對，反正他們就是會有一個工作，是專門去幫你捏奶的。如果說奶卡住，或者說有可能有乳腺炎的一些什麼前兆之類的，他們都可以幫你解決，非常專業。那這種奶子專家——姑且就叫奶子專家——他們的出勤一次的費用，就我理解是三到四千塊左右。就是說如果你在家裡，你要打電話請這些奶子專家到你家的話，是三到四千塊。當然你沒有遇到是最好，但如果說你有遇到的話，像我太太就是三到四天可能就需要她來幫忙一次，所以你要把這個費用也攤進去，就算大概一天是一到兩千塊。

然後再來就是你還要去算護理師 24 小時 on call 去服務你，你需要的時候就可以把小朋友 call 來，那如果說你覺得累的時候就可以把小朋友 call 回去，那他就會幫你餵奶、幫你照顧、幫你換尿布，然後教你洗澡、一些什麼衛教的東西。那像這個東西的話，就我姑且去算它是類似說像哈瑪多龍會在家裡找一些保姆嘛，這些保姆其實目前的價格都是蠻貴的。我們用全職保姆來算的話，最低價的應該是那種跳機移工了，當然這個不鼓勵大家做，只是我知道身邊有看到這樣子的狀況，跳機移工大概是五到六萬塊。那個就是說，假設抓到會出事情，不推薦大家這樣做，但是就有人這樣做。那如果不是跳機移工的話，就是直線上升，甚至會到大概十萬塊左右。那他不幫你做其他事情，他不會說什麼在家裡還要幫你煮飯、幫你收熱什麼，沒有，他就是只照顧這個小朋友，然後就需要這樣的一筆錢。

所以其實你把所有東西都加總，然後去把它攤成一個月的話，你會發現沒有很貴。就是我們不是講總價這個東西在這個價值觀裡面是貴還是便宜，總價三十幾萬一個月一定不是便宜的數字。只是說，如果你有這些服務的需要，你把它加起來，它其實是不貴的。所以會覺得說當然這個東西是要夫妻去討論。我覺得不要看太多那種網路的東西，然後去受到很大的壓力，因為有些夫妻就真的覺得說他們生小孩是很輕鬆的，所以他不想要花這個錢，他會拿這個錢去再做一個什麼二度蜜月旅遊，那很好。如果說就是有需要的人的話，其實這個價錢算起來也不會是太貴。在我自己的認知，仔細地去盤了一下，發現是一個 CP 值蠻高的一個選項。

但其實大家都要自己去理性地跟伴侶做討論，然後去做選擇，盡量不要看太多那種網路的文章。因為像有些網路文章我覺得會給大家很大的壓力——你沒有做什麼事情，你就不要生小孩，或者說你就不要結婚，你就沒資格做什麼。那都是那種網路魯蛇廢物最喜歡去網上貼這種東西，正常人——我沒有看過那種真的是正常人，然後生活條件過得很好的，會有空只能去網上指教別人，真的是比較少。所以盡量不要去受到這種東西的壓力。因為如果你真的是要把每個網友講的一些大話全部都加進來的話，那你真的這輩子會過得很辛苦。一堆東西都是按照大家的標準，你每個都要達到的話，那你什麼事都不用做了。

所以我覺得還是要去評估自身能力。但如果說像我們這種狀況的話，就是我們真實的體驗跟大家分享，我們就覺得說這個東西其實是還蠻划算的，仔細算一下是這樣子。因為後來回林口之後，我們也會有類似的問題要去做解決，那就後面如果說再有一些這種有趣的預約的東西，再跟大家分享一下。

好，那接下來我們就來看一下市場的最新動態。那是如自己所盤算的，就如果是貝森特持續在講話的話，市場就會是一個比較穩健的狀況。那我們在過去的一兩個禮拜看，就真的是這樣，因為盧特尼克跟納瓦羅比較沒有發話的空間，所以呢就稍微的好一點。但後來盧特尼克又要開始出來講話了，所以大家最好皮繃緊，後面搞不好又要再去製造一些人為的波浪出來給大家了。

那在關稅方面呢，其實現在在美中這邊都開始出現了一些退讓的環節。但是我還是鼓勵大家不要用「跪」這個字，因為很多人都喜歡用跪，就像說誰做什麼事情、另外一方就是贏家。其實不是這樣，它是一個拉扯，它是一個談判的過程。那目前的狀況就是說，在美國這邊的一個退讓是在一些消費電子跟伺服器，在過去有一集把詳細的稅率跟大家分析跟討論過了，這個就是美國方面的一些退讓。然後在汽車方面呢，就後面可能會出台一些政策來保護國內汽車產業，要因應因為海外的進口的成本變高了的問題，可能也會有一些新的做法出來，這個貝森特有提到。

那在中國這邊呢，其實也是出現了類似的一些做法，像是在一些半導體的元件上面，一些 discrete 這種分離式的元件上，中國這邊都是開始去把那個關稅給降掉。為什麼會這樣做呢？因為這個東西是他們沒有辦法自己生產的。所以會注意到說，美中這邊都還是有維持一定程度的理性，就是當今天某個產品是他們自己沒有辦法生產的，或是跟自己的利益是高度綁定的，那他們就會選擇去對它做豁免。這個是一個我們在市場上希望看到的訊號，就是它不是已經變成是一種那種全面熱戰的關稅戰的感覺。所以只要還有一些理性存在都是好的。

然後再來呢，就是最近有一個蠻大的新聞，就是白宮的發言人去針對 Jeff Bezos 跟 Amazon 做出一個打擊，說他們在這個商品的頁面上面據傳要放上「關稅」這個欄位，直接告訴大家說 import charge 是多少，就是關稅的成本是多少。然後這個東西他認為是一個 hostile，就是比較有攻擊性的做法，然後更是政治性的做法。

而這個其實還蠻有趣的，因為在我們海外的角度來說，你去買 Amazon，其實上面就是會標說 import charge 是多少，也就是那個進口的稅率，包含說像是各種關稅或是相關的費用都會標在裡面。那我自己因為在台灣跟在歐洲都會使用 Amazon，所以這個對我來講是一個很稀鬆平常的事情。但在美國這邊，其實他們是把相關的稅率是包在價格裡面，就是大家其實不會看到一個 import taxes。所以只能夠說，亞馬遜並不是要針對川普去做打擊，他只是因為可能在後面關稅的成本是真的會高到會影響到商品的價格——也就是他不是可能只是簡單的 10% 可以解決，甚至有些商品搞不好因為這樣子會翻倍貴，所以他不想要去扛所有的壓力。就大家會覺得說「這個臭奸商，那邊漲價，這個東西之前是多少錢，現在是多少錢」，所以他們可能有這樣子的規劃。

雖然他們隨後就是出來否認，說沒有要這樣子做，我覺得是受到一些政治的壓力。但本來有這個討論，我認為應該是想要去避免就是大家把這個錯怪在亞馬遜身上——是不是你要削錢，你要趁機漲價？他就等於說直接把它分別列出來。美國這邊本來不會列，然後現在就是跟海外的亞馬遜的一些據點一樣，就是你要買東西的話，他會把關稅是額外列出來給大家看。所以他有這個做法，就想要這樣子搞。

那只是後來就是可能變成是一個政治的博弈，那川普就馬上出來要去針對這個事情發表意見。雖然他後面說打了電話之後就覺得沒有什麼太大的狀況，但這個其實也是證明說川普有很高的理智。如果一些消費品的通膨產生，對他來講應該是一個超大的壓力。等於說他在這個事情的表態上面，特別是這麼立即快速地去做回應，就是他們應該也是非常的緊迫在盯著關稅可能會造成的通膨影響。

雖然我還是覺得，關稅一定會造成一定程度的通膨，那個就是後面是必然發生的事情。可是至少你可以確定的是，他們也很怕這件事情發生，所以是好的。就不是像可能本來蠻多人的期待是認為說，他應該是完全放推了，他想要把經濟砸掉。他應該是有很多的理性在背後，所以即便他做出一些非理性的行為，然後現在在過去的差不多半個月、一個月是拼命地去做修正了，開始有一些放軟的跡象。不過呢，大家可以認知到的是，至少他沒有瘋。他只是看起來瘋瘋癲癲，但他沒有真的瘋掉。他還是有蠻多的軟肋在那邊，這個對我們的市場的投資來說是一個很好的事情。
