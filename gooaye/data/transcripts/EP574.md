運動是要給幫助身心的調劑，知識衛星要帶大家一窺瑜伽的美好。《尼泊爾正宗瑜伽課》是目前台灣學員數最多的線上課程，已經超過了三萬人報名加入。尼泊爾國家級大師打造了結合體式、呼吸法與冥想，一次鍛鍊你身心靈的課程。現在人生活忙碌，大家都在追求身心平衡，或許從這堂課入門瑜伽就是一種選擇。

課程貼心地附有中英配音，跟練不用怕聽不懂老師在教什麼。內容有 20 分鐘到一個小時不同的版本，可以跟著自己的瑜伽好朋友一起對照著理解，就像是我們在學習其他運動也會找 YouTube 的一些大師示範說明來幫忙。不論你是早上要醒腦、下班想舒壓、睡前要放鬆，都可以找到對應的單元練習。課程從基礎的瑜伽理論開始，到比較進階的呼吸法跟導引練習，甚至還有增強性魅力的助性瑜伽，協助送幸福。

如果你有興趣培養瑜伽運動的好習慣，不用跑到喜馬拉雅山腳下，這次知識衛星特別把老師帶回台灣，讓大家體驗發源地的正宗瑜伽。部分章節已經上架，跟著家人一起開練，鍛鍊身心靈，充實家庭和平和樂而不為。課程目前早鳥限時優惠 45 折，結帳輸入專屬的折扣碼「gooaye yoga」（G-O-O-A-Y-E Y-O-G-A），還能夠再折 350 元。在此提供給所有有興趣的朋友們。

昨天晚上跟幾個快樂好朋友一起吃飯，都是地方產業界的資深美少女跟資深少男，真的是每次跟他們出來都會覺得學到很多東西。當然，其實到最後已經比較不會是像蠻多人可能會想像的，因為我記得跟大家分享過，自從這個節目開始，我算是獲得了蠻多資訊的優勢，但這種資訊的優勢，其實不是每次你跟人家出去吃飯，或是跟人家開啟一個對話的時候，就問說「你跟我講什麼東西？」一般本來都是很自然的發展。

甚至到後面會發現，我相信大家如果是在地方各種群組有去埋伏，不管是什麼股票群或是自己產業圈的，就會注意到，那種越是高端的圈子，就是他們知道最多東西的，他們越常講的是那種很日常的東西，每天跟你早安晚安什麼的，不會說整天都在交流一些非常產業面的數字之類的東西。這種東西可能是比較偏向說，如果我注意到一些變化才拿出來講。

那這些變化其實不像是有時候我們去做一些動能股，會整天去一應公司的一些發展，一下有單、一下又沒單、一下誰又搶單、一下什麼良率又怎樣，整天改來改去。但其實到後面會發現一些大趨셔的形成，它是長期的，它絕對不會說什麼今天 AWS 要去推一個伺服器，明天又不退了，後天又要減量，大後天突然又增量加單。那個就是說，短期一應股價的人會整天在搞這種消息。但如果說越是發展到那種比較頂級的圈子，其實就會注意到，早就跟你說 AWS 好，好就是好，那過程那些震盪就不重要。

所以其實不會整天 focus 在這個，反而會花滿多心力都在聊生活中的一些東西，育兒啊、怎麼樣的平民紅酒啊、或是茶類啊，再來可能最近又去哪裡玩，什麼都比較平常這樣子的東西。那其實我還滿喜歡這樣子的一些對話，因為到後面我就發現，如果說你想要去達成一個可能更高的成就，或是你想要成為他們的人、想要跟他們一樣強大，或者你想要到他們差不多的年紀時，你也有那樣的一個狀態，你就要學一些人家的一些生活態度跟做人處事的一些道理。那最後會發現說，這還是形成一個人最重要的東西。

那些所謂的資訊或是一些消息，它只是在過程之中可以去輔助你的，但是真的是人格特質，跟做選擇的一些判斷能力或判斷品質，那個才是真正會去影響最後你發展到底是好還是壞。那這些長輩們，我覺得他們滿有趣的地方是，他們跟我自己身邊的朋友，其實年紀都比我大一些，但有些可能就跟我一樣，三十幾快四十歲，那有些可能就五、六十歲、七十歲，比較少有二十幾歲的。你會發現其實每個階段的朋友，他們給你的一些生活上的指引，或是一些當你跟他分享某些事情，他們給出來的 feedback 都會完全的不一樣，這個是非常有趣的一件事情。

所以我也是很享受於這種衝突感，在某些圈子他們會告訴你一個答案是怎麼樣，你會發現這個答案已經像是一個共識，就大家都這樣子想；可是到了不同的圈子之後，會發現他們的想法是完全不一樣的，但他沒有誰優誰劣，可能每個人的人生階段不一樣，他做出來的選擇就不一樣。或是這些看起來是長輩的人，他們如果回到二、三十歲，他們其實講的東西就跟現在我們身邊二、三十歲的朋友一樣。可是，有些東西真的要等到一定年紀後，才會開始慢慢的理解。

所以我覺得有時候不免在開始想說，會不會以後我們也變成小朋友討厭的長輩？就跟方案說，你一開始是不懂長輩，後來到理解長輩，然後你就變成長輩，就你也長輩化，每個人最後都會不免地長輩化。但當然我們在長輩化的過程之中，會發現其實每一代都在做加強。像以前我們家在發那個過年紅包的時候，我們家裡的一些叔叔伯伯或是自己的父親，他們就會講說，我們不要浪費時間在那邊吃飯，拖延這些小朋友的時間，你看他們都是想要趕快拿到紅包就出去，所以就趕快發給他們直接解散這樣。那他們一定也是說，他們的長輩以前搞不好就是會囉囉長，會發表一些感言。像我小學畢業的時候，媽的一堆有的沒有的，什麼政府的達官顯要、校長，一堆話要講，一上去講十分鐘起跳。那其實發現老師也會迭代，對不對？新的老師發現說，以前那種長輩老師那樣被罵超多，那新的老師要去討年輕學生喜歡要怎麼做？就是「我不會講太多，一分鐘講完」，這講完就會獲得如雷的掌聲。

所以其實大家都是慢慢地迭代，只是你會最近還發現說，你的年紀變大，你還是會不免地沒有辦法跟上最新的東西，所以你的小朋友就還是覺得你是一個法金長輩。那你在那邊講話的時候，他可能就想要回你一兩句，就像是當時我們會去回我們的長輩一樣。那我覺得沒有辦法，那是一個階段性的變化，但最後可能真的大家都也慢慢地長輩化了。

昨天聽他們分享的一些我覺得比較感動的東西，都是關於他們小朋友的一些成長。因為他們年紀就比較大，小朋友可能大學畢業，或已經開始有一些發展，或正在念大學之類的。他們就會分享說小朋友，有些是像他從小可能不一定喜歡爸爸喜歡的東西，可是他發展出自己的路線；然後有些東西竟然跟爸爸還是有一些共識，覺得說爸爸是看《魔戒》長大的，小朋友竟然也很喜歡看《魔戒》。都已經這麼久了，這個小朋友竟然還可以找到這樣一個我真的覺得是曠世鉅作的東西，現在已經很難拍出這樣子電影，然後小朋友竟然喜歡，而且他可能每個月都要看一次。

那這個小朋友可能因為跟爸爸找到一些共識，有時候可能小時候會有一些衝突，小朋友就會覺得爸爸管太多什麼的，可是其實爸爸也曾經年輕過，也想要找一些方法去跟小朋友互動，直到可能找到一些共同的語言之後，大家開始互相理解。小朋友就推爸爸聽饒舌，爸爸說從來沒有聽過饒舌，但聽完他介紹的饒舌，覺得「真的是超棒的音樂，我要推薦給大家」，所以我們也跟著他一起聽了饒舌。我們就長輩，每天聽所謂的年輕人的饒舌，不然我們就只知道 Eminem、阿姆之類的，或什麼 50 Cent 之類的。

那再來又有些小朋友可能是很喜歡打牌，就是喜歡打《遊戲王》卡。《遊戲王》卡其實是我們小時候的一個回憶，沒有想到現在的小朋友，高中、大學生，他們竟然也會玩《遊戲王》卡，然後把規則搞得非常的清楚，這是他的興趣，他會形塑他自己的一些判斷或什麼的。所以有時候甚至這樣，就是從本來你是他的監護人，你要去照顧他，當然到長大都還是，可是因為他長大之後，他會有自己的一些個性產生，他會有他喜歡的東西、討厭的東西，或是他的一些價值觀。那這些東西可能最後是反向影響到你，然後讓你可以出去跟朋友分享，我覺得這是一個很酷的事情。

他們同時也就是叫我趕快多生一個，而且其實他們講這個時候我不會覺得，不像以前小時候，我們聽到人家講說叫你生小孩，你就覺得反感，超大的「干你屁事，老子要怎樣我自己決定，我身邊朋友哪一個生小孩？」那可是聽他們講就覺得，因為畢竟第一個，他們自己都以身作則，自己都有滿多小孩的；第二個，聽他們講這些故事，你就可以理解說，他們講這個是一個來自真心的感嘆，都不是說因為他的爸媽叫他這樣做，所以他就叫你這樣做。不是，因為他可能以前也是曾經反骨過，覺得說不要、那麼累或什麼的。因為確實就是累，像他們有分享，小朋友一開始，他第二胎要生第三胎的時候，他嚇到趕快去把輸精管綁起來，再繼續生下去是養不起，當下都會這樣覺得，當然現在看就覺得「根本還好」，可是當下其實壓力都很大。

因為其實我還是覺得，我們人在二十跟三十幾歲的時候，其實可能是我們最辛苦的一個階段。這個其實看一些海外的 Life Lesson Coaches，他們也是講類似的東西。因為這段時間就是你正從大學畢業出來，要去找工作、要去經歷很多的磨難，那同時又沒有什麼錢，又需要買房、要扛車貸，突然間又有小朋友什麼的，你所有最大的壓力都是在這時候產生。可是同時，有些事情也就是你就在這時候去佈下種子，你才可能在後面會有一個可以去享受成果的機會，因為時間過了，可能有些事情就是沒有辦法做。

所以其實最後面大家都要去仔細思考，你的人生要的東西到底是什麼，每個人其實要的東西都不一樣。那像我去聽他們講，我就不會覺得他們在跟我宣教，我就只會認為說……以前會覺得「這個就長輩在宣教」，可是現在就會覺得說，你知道他們就是把他的感嘆跟你說，他真的覺得這樣子。如果說你不認同怎麼辦？其實你就是微笑點頭就好。所以如果讓我回到過年圍爐的飯桌上，那個長輩去講一些我們講的所謂的幹話的時候，如果現在我回去坐在那邊，我就是會點頭，然後我會問他說為什麼他會這樣覺得，就不會像以前年輕的時候就想說「干你媽的到底干我屁事」，有些東西就會開始慢慢地去轉變，然後就會慢慢去……所以最後面真的是從反對長輩到理解長輩，然後變成長輩，可能我們的路線都是這樣子，只是我們變成一個更好的長輩，差不多是這樣子。

接下來我們來進入市場的話題。

我們上集是有聊到說，關稅對台灣的影響應該就會慢慢降低，因為當一些東南亞的國家開出來，或者說假設美國跟墨西哥最後面也是沒有什麼太大的問題的話，其實台廠是有「狡兔三窟」，滿多地方都可以出貨到美國去。然後再來就是美國本身可能還是有一些豁免，像現在對 AI Server 的一些豁免等等，若那一段落實的話，其實我相信影響的程度是會越來越低。所以這個我們過去認為是一個蠻大利空的東西，現在就已經是慢慢漸趨平緩。

另外一個我認為是很大利空的，也就是匯損的問題。最近在大立光的財報開出來之後，我相信應該蠻多的跟我一樣的想法，就是會不會其實台股真的不用跌這個？因為大立光那個匯損是蠻大的，它其實本來好像預估大概 35 到 40 塊之間，然後這邊開出來 7 塊，不見得這個錢是跑去哪，就是匯損直接賠了四分之一。所以對大立光來說，它算是遭遇了一個很大的匯損衝擊。

那這匯損衝擊最重要原因，當然是因為首先我們那時候升值真的太快，如果是一個慢慢的升值趨勢的話其實還 OK，因為會造成一些虧損，但除了你帳上的一些資產之外，再來就是我可能在過程之中，我報出去的價格是固定的某個美元售價，可是台幣一直升值，那這邊收到錢、我要去買料、去做一些調整的時候，就會注意到你那個利潤就是全部都噴掉了。所以這是有可能會發生的一些實務上在工廠面的狀態。

所以我們會擔心這種很快速的升值，可能會對財報造成的一些影響。那現在在大立光開出來之後，其實它是直接反漲的。當然它在前面過去幾天有先跌，有時候市場就是有人會先知道，所以說不定這就是一個反彈而已。但是其實以我們的判斷來說，像吃掉這麼多的一個 EPS，然後竟然還可以……但前面有先跌一小段，可是它確實是上漲的，那可能是代表說，有蠻多人都已經有做好這樣子的一個心理準備，也就是說或許後面連續開匯損，它並不是一個需要擔心的事情。

所以有時候跟大家說這個市場是動態的，我們都會慢慢去做調整，但是就是都有一個 but，but 就是我們過去也看過很多，其實台股市場是一個……他一句我朋友的說法，真的覺得講得……台股市場是一個「不見棺材不掉淚」的地方。就是有時候當然一個利空開出來，它真的就是不跌反漲，然後就讓大家覺得說這個東西應該是沒事，全部人衝進去。然後最後面的第二家、第三家、第五家、第十家、第十五家，就是每家都連續開……以現在的案例來說就是匯損，第一家開，大家覺得沒事，衝啊，趕快去買；然後之後面連續幾家開出來之後，就是後面大盤的下檔，然後就突然間狂崩。這個是我們在台股很常看到「不見棺材不掉淚」的一個行情，都先撐，撐到後來撐不住的時候就直接一波，給大家爽歪歪這樣。

所以我們還是不會去把匯損這個問題給 rule out 掉，因為它確實就是會對我們去分析公司造成一些影響。就像以大立光的案例，你知道大家估三十幾塊、四十塊，然後後面你只剩下七塊，你說你的 P/E ratio 或是我後面可以給你一個乘數跟估的股價、TP，還是一定會到那個地方嗎？不一定，因為數字就不一樣了。那接下來，這個骨牌效應，就是後面還有一堆公司都會面臨一樣的狀況。但是如果說我們連續地觀察到一些……就是說以台系廠商來講，大家會盯著一些重點，像一些交換器或一些 Server 的 ODM、OEM，如果說他們也把匯損的成績單開出來，然後都不用去反應的話，那真的可以相當程度地視為說，匯損這件事情可能已經不存在了，我們就視為說市場已經是完全去消化掉這件事情。

所以這個是我在後面的一個，我認為蠻重大的一個調整。因為其實我過去一直認為說，首先第一個是關稅，再來就是匯損的問題。那關稅已經在上次，因為我們知道說有一些台灣的口岸，就是可以出口的地方，他們看起來是還 OK 的，所以它是有一個地方有機會可以閃的，那對我們來講，它的利空的威脅程度又降了很多。然後再來就是像大立光的財報開出來之後竟然是反漲，這個對我們來講，也是我可以視為說，說不定市場已經消化完這件事情了。

所以以 AWS 的角度來看，我覺得 AWS 目前的狀況都還是非常好，它算是在 ASIC 陣營裡面非常強大的一個存在，所以應該是沒有所謂的砍單狀況。如果說你有看到一些數字的調整，可能比較偏向是在它的代數的轉換上，因為要開始往更新一代去做佈局，會有一些數字的調整。就像是有些人去追客戶端的數字也會發現說，為什麼會有那個數字的跳動變化，那可能就是產品組合的配置修正而已，那不會是一個太大的問題。所以相關的供應鏈以台股來講的話，就可能像智邦、貿聯、高技等等，就是從可能板子到交換器、compute rack，然後再到可能一些線材或者說一些 power，我們目前應該是沒有看到有大修正的可能性，至少在短期內是這樣子，這邊稍微跟大家分享一下。

那在 AWS 伺服器有一樣，就是他們最近有蠻多話題的，它推出了一個叫做 IRHX 的東西。那是什麼？這就是說 in-row heat exchanger。那這個東西是幹嘛？這個就是……講白了就是我們目前看到的伺服器，特別是假設你有在做台股，應該很多人的概念都會認為說，所有的伺服器應該都在走水冷吧？但其實不是這樣子。最主流的伺服器跟 Data Center 的一個建置，目前其實都還是以氣冷為主。那走到 AI 的話，因為它的一個 server 要散的熱是太多，它那個瓦數非常高，像現在的 GB200 就是 120kW，這是一個非常高的瓦數，所以這是一定要走水冷。

但是我們講的水冷，其實是比較偏向說最主要就是在這個 rack 裡面，就是用水冷去跑，因為它是要跑那個 cold plate 去做散熱。可是拉出來之後還是有各式的玩法，有些是用水對水的方式，就是用 liquid-to-liquid 去做解熱；那有些是用 liquid-to-air，那可能就旁邊放一台 sidecar，然後這個 sidecar 裡面就可能用風扇去吹這樣。

那 Amazon 這個 IRHX，它對於市場會有一定程度的影響，是因為首先它拒絕了……這樣看起來應該是它拒絕了那種成套的 solution。所以就像 Vertiv 可能有一個成套的 solution 可以供這些 Data Center 或客戶，你來跟我買，我就是一整套的 solution 直接給你，然後讓你可以去用在你現有的一個 Data Center 裡面。但 Amazon 不想做這樣的一個事情，當然他們官方的說法，因為有公布影片，是講說因為他們既有的 Data Center 可能它是以氣冷作為一個核心的，所以它還是希望說可以跟氣冷的的東西去做結合，那這樣子我就不用去重新設計我的 Data Center。

但這邊其實不會改變未來的一個趨勢，所以未來的趨勢就是說，新建的 Data Center 其實蠻多都會直接是以 liquid cooling 的角度去做一個思考。所以水對水可能就是未來的一個主流，就是在新建置的上面，包含像是樓板的承重或什麼，它就會去直接往那個方向去做。那可是舊的一些 Data Center，它假設要導入的話，那為了去跟現有的東西做結合以及不要浪費，那它可能就會弄出像它這種 IRHX 的一個解方。

那具體的做法其實就大同小異，就說概念上其實都類似，是取的不一樣的名字。像一般我們在業界這邊是講說 CDU 嘛，那在這邊它就把它拆成是 water distribution cabinet 跟 pumping unit，但其實道理一樣，就是說去分配這個水流去做調節，以及給它一個動力把它 pump 進去，然後跑到 rack 裡面去 cold plate 散熱之後，再把它倒出來。那倒出來就是送到 fan wall 那邊去，就是用風扇去吹它。那只是說可能一般是我會再搭配一個風扇的 sidecar 或是一個散熱的 sidecar 在旁邊這樣，然後現在就是把它變成一個統一化的東西，在我們自己設計的，直接跑到這裡面，然後藉由可能一整排的風扇的 rack 去吹，然後之後把熱帶到後排去，然後再散到外面去，這可能是他們的一個解方。
