## 瘦瘦針進化論：Ozempic → Mounjaro

最近在運動上面獲得很多新的啟發啦。有時候我覺得也是需要有一些像地方的聽眾給我一些認證，然後我就又再一次有動力——其實我就是一個蠻膚淺的人。最近有一個地方的兄弟在健身房跟我合照，那後來這張照片他發給我轉發，蠻多人就講說「幹你真的變超級瘦、你臉是尖的」什麼的，我就覺得心情特爽。然後就覺得接下來我也可以再回去去拉個幾下、或是去打個什麼網球之類，突然覺得自己充滿了活力啦。

我覺得瘦肉針對我的幫助真的很大。最主要並不是說它真的限制了我多少的食慾，它就是給我一個精神跟心理上的 cue，讓我知道說針都打了所以你就要好好的振作起來。所以跟自己身邊朋友相比，蠻多人針打下去之後他是吃不下，喝酒就會吐或是說可能腸胃不舒服，然後根本就沒有力氣去運動之類。我算是相對的幸運，食慾都是正常的，只是食量變小一點。

那經歷過了 Ozempic 到現在的 Mounjaro，狀態都差不多。Mounjaro 一開始打下去的時候——因為我前面 Ozempic 我是不知道就直上了嘛，那後來發現說這樣子有可能什麼猛爆性胰臟炎、急性胰臟炎之類，就嚇到，知道說那下一針我就要乖乖的。所以 Mounjaro 一開始是打 2.5。然後 2.5 打下去我立刻就知道出事情了，為什麼？因為 2.5 的 Mounjaro 根本就無法擋住我的食慾。所以我本來以為說可能差不多快到目標了可以直接停針，但我現在發現說可能不能夠直接停針，因為如果停得太快的話——那時候 2.5 的 Mounjaro 就讓我整個晚上都在吃東西、狂吃，然後吃到那時候來我們家的客人都說「你完全看不出來是有在打瘦瘦針」。我說因為我剛換針，那個劑量可能是偏低。

所以呢，當然醫師講說你最好就是隔一個禮拜，然後再去把它加大，甚至是最好你把整個流程走完再加大。但你就知道說，我是不會去做這種事情的——這絕對不是什麼好的示範。反正就是我就去問了醫生，那我們也不要講是哪一位醫生，反正就說最快多久可以補下一針，那我就在那個最快的時間就立刻再補下一個，我就直接打到 5。因為我發現說媽真的沒有辦法，那個劑量太低了，我的整個飲控可能因為這樣會出問題，所以就目前還是得去依賴這個藥物。

我覺得藥物它除了是精神上的 cue 之外，就是它去限制那些食量對我來說是好的，目前整體的身體表現也變得更好。我覺得說因為我瘦的最主要都是脂肪，所以我覺得睡眠品質或是所謂的發炎的感受你也感覺得到好像是有降低了。每天痠痛的時間點是變短，或是起床背痛的機率是變低了。所以我真的覺得瘦下來是一個蠻不錯的事情。

## 機能性訓練：重新認識身體

最近在運動上面也獲得蠻多新的啟發。非常感謝我的這個長額像 Ryan 的教練，我非常的強大，他教我那種機能性訓練，或是有些人稱為是功能性訓練，那他真的是有感的讓我可以去支配我全身的各個關節。就是你從來都不知道說原來你全身有這麼多的關節，然後把他們湊合在一起的話，就比方說一個揮拍的動作，可能是從腳底板開始發力到你的小腿、到大腿，然後最後面到你的手。當然這個我們去學泰拳的時候教練也會教你，可是我覺得沒有把每一個動作拿出來雕的話，就是你還是你以為你懂了怎麼樣去控制這一些關節或是你的肢體，但實際上你可能還是不太懂。

他教我的訓練其實看起來有些就覺得好像是「欸他媽太簡單了吧，就起立蹲下」，可是那個起立蹲下會蹲到你全身都噴汗，甚至是不用負重就可以這樣子。然後再來就會玩一些奇怪的道具，什麼用 TRX 的繩子去做一些東西，或是拿一個像什麼奇美棒的東西去做伸展或什麼的。一段時間下來就發現，真的是懂什麼叫做用髖發力，也知道什麼叫做用髖去移動。覺得說你坐在地板上然後屁股挪來挪去你就可以前進後退之類的，那對身體的掌控度變得非常好。所以他可能不像平常的重訓師針對肌肥大去做訓練，他是針對一個全身的協調性，可是我覺得那個感受是極好的。

那在他跟我分享蠻多觀念之後，我就反問他一個——因為我剛好在一些報紙上面有看到說，像那個三本柱神（譯按：山本由伸、大谷翔平等日本職棒明星），他跟一些就是外籍球員跑去美國不太一樣的點，是蠻多都是去美國做所謂的科學化訓練，整體都覺得生活品質變得很好。但我更不確定的是，搞不好是因為現在外面行情真的太好，所以我心情就很好，說不定接下來回檔回一個月的話，那我就會感覺我全身又很沉重，這邊痛那邊痛。所以因為現在太多事情發生得太快，我沒有辦法明確的去識別說到底是因為什麼變成是這樣。但整體來說我覺得，那時候去打瘦瘦針那個決定是一個非常好的採發，就是好像說把整台老車給發動起來了。

跟我教練聊的時候我就講說，因為他那時候會去提到說有些什麼球員的生涯的高峰可能是在 20 幾歲，然後最後都是一路走下坡什麼的，所以我們要盡可能去維持自己的體能之類。我說沒有，我生涯的高峰應該就是現在，因為我以前根本就是沒有什麼在運動的。後來在聊天的時候是有聊到，就是因為他跟我一樣是在特戰部，就是我們都是傘兵，那其實傘兵那時候體能也是蠻不錯的，因為會被操到爆炸，整天要跑步、拉單槓或什麼的，但感覺也沒有現在好。現在整體的體能狀態可能是我人生的巔峰，所以蠻多人的人生巔峰可能是在 20 幾歲，然後我巔峰竟然是發生在可能 30 幾歲然後還在上升中，還蠻高興的。

就前面好像說「你有一副好的身體沒有什麼狀況你都沒有在用它，你整天都坐著打電動或是看影集什麼的」，大家終於有去用自己的身體，而且知道哪邊可以發力。然後就會注意到說就是他的一些動作他教你怎麼發力、用什麼髖關節去傳導什麼，那個動作他做起來真的上馬異常的帥。這也是為什麼那時候我會在健身房發現這個教練的主意——他做一些動作可能看起來蠻中二的，就是不知道他那個發力的動作很像那種天堂的死亡騎士變身那種樣子，然後就這樣勾下去然後揮刀之類的，就是那個動作整個是流暢超級帥的。那就是有辦法去把你的肢體可能調節成那樣，雖然我現在做還是有點像智障的感覺，可是慢慢的有一直去走向一個更好的方向。

所以呢，我的心得就是我認為減重是很重要的，然後再來是——以前都覺得說可能就是有氧加重訓，沒有想到真的是偶發性的看到教練在做超奇怪的事情，然後去上課之後才發現說機能訓練或是說這種功能性的訓練，它對身體的幫助可以這麼大。然後你可以去發現說以前我去做這個角度的動作我身體會不太舒服，原來是因為我的髖或是哪裡卡住、骨盆卡住，那現在把它打通之後，其實整個人的柔軟度彈性變很好，它會讓你很想要就整天跳上跳下，你會很想要一直動，因為你好像說重新認識你的身體，然後你的身體它變成是一個有趣的東西。差不多是這樣的一個感受。

這邊跟大家分享一下最近自己的減重跟健身的小小心得。因為現在開始有些新的聽眾加入，我最近有看到說以為這個是紅酒節目、他以為這個是瘦身節目——其實這個節目就什麼都不是。我們就是隨便在聊天，以前可能目標導向會很明確，我就是要交市場的朋友、要聊市場的東西，我現在都沒有在管這個。反正我們有固定的格式，就前面一些老聽眾可能跟我有共鳴的，他們聽就有感覺；新來就想說你到底在講什麼。但有些人也是因為這樣，新來的一聽就說「我就是喜歡這個節目，開頭會跟我聊個天或什麼的」，反正不管，我們不會去修改我們的格式。歡迎這些新來的聽眾，因為現在盤排很好，所以又有更多人跑去開戶了。

總之就希望待後面都可以順利的度過，不管是噴上去的時候你心情可以平復一下——有些人就會賺得比你多之類的——或是說掉下來的時候大家知道說「自如常擾」，這個都是蠻重要的。

## 被動元件大爆發：國巨陳泰銘 AI 日

接下來我們來進入盤上的話題。我們在過去已經跟大家一直聊到說，就是被動元件這邊在後面一定會有一個缺貨的情形產生，只是當然他的那個時間點是不好抓。最近就是 PChome（譯按：應為國巨集團）—— 也就是國巨的陳董陳泰銘，他搞了一個 AI 日，然後在上面大吹猛吹一頓，整個國巨走起來，相關的一些被動元件都全體的轉強。今天也看到信昌電有一個出貨交期拉長的通知信，它並不是一個漲價信，可是市場就很買單，就直接給它一個漲停。但出貨的交期拉長當然就可能代表說後面就有機會會看到一些漲價的可能性。

所以這個也算是一個例子，就是為什麼還是要去研究一些基本面。這個基本面可能有時候研究下去——像前面講說「啊這些被動元件都不會動」或什麼的，海外的也是有動，台灣的沒有動，可能就只是時間還沒到。如果說這個基本面是正確的話，那時間到的時候它就是會反應，只是你沒有辦法知道說反應的時間點是怎麼樣。所以當然有些人就會轉向說「那我就全部都做技術面」，但技術面的話你就是有可能會買到一些是假突破的東西，或者是市場資金在亂炒的，那就變成你整天要停損。所以它會變得有一點反直覺，就是大家的直覺都會是想要去趨吉避凶、不會想要有太多的停損。當你的停損它佔比是很大的時候，那它也不是機械化去操作——就像說你有程式去做它的話，其實我覺得你的心態會跑掉。那這就是為什麼你還是要大概知道一下你買的東西是什麼。

針對被動元件的評論，因為在過去的數集、已經數個月了，都已經在稍微提到，我們有看到這個產業趨勢，所以在這個股價表態的時候我們就不會再去多說什麼東西。但就是確定說，不管是鉭電容、或者說鋁電容、C-cap、IC-cap、然後跟 MLCC，其實都是有注意到這個供需是開始越來越緊。特別是高階貨——高階貨是超級緊的。

那高階貨的話呢，台系可能就不一定碰得到，就是要把麥克風交給不管是韓國的三星電機，然後或者是在日本的村田，他們是老大。那台系的話他們受惠的故事比較偏向是，當然他有部分可能是切進去這些高階的 AI 伺服器，還是有，是佔比可能沒有這麼高。那另外一部分就是說當大家都轉向高階 AI 伺服器的時候，有沒有可能就是出現一些訂單排擠的可能性，那這些訂單排擠會不會讓這些小弟們就吃得很飽。這個其實跟 2017、2018 那個敘事有點像。當然我們不是要暗示說這個會跟 2017、2018 一樣變成那種超級大表股，只是可以去密切關注後續的一些發展。

那有些發展可能就是在起頭的時候你不一定看得出來到底是什麼，就像說像我們前面在跟大家在聊 CPU 的時候，我們最早其實也只是看到說奇怪「美股崩成這樣，為什麼美國的 CPU 類股都很強？」，那因為這樣去做佈局嘛，其實最早真的是因為技術面的角度進去。後來發現這個 agentic workflow 裡面它真的會用到很多的 CPU，當我們確認這樣的一個產業趨勢之後，你就是會更有底氣的去抱住手上的一些部位。到現在大家已經知道說 CPU 就是市場裡面的當紅炸子雞。

## AMD 法說：Server CPU TAM 上修

不管是 Intel 開的一個財報或是電話會議，或是 AMD，大家都在關注這個重點。AMD 這邊在法說會有一些重大的訊號——他們去上修了 server CPU TAM，從原本的一個估算調到 2030 年（譯按：原文 2023 應為口誤），他們是估 120B，年增是大於 35%，這個速度是很快的。同時他們也講出說他們有信心可以拿下 server CPU 超過 50% 的一個 market share，那這個我是覺得機率是還蠻高的。再來就是說 AI 的營收能見度是變得很高，2027 年 AI 的 datacenter 營收會到 tens of billions，也是一個非常高的一個數字。同時也呼應到 Intel 法說裡面所說到，就是 agentic AI 跟 traditional 的工作負載結構性的推升了 CPU 的一個需求。

所以簡單來講就是 CPU 加倍變市場中的一個最重要的核心，那其他相關的一些公司也都跟著動起來。

## 台股 ASIC 全噴：創意 + 世芯

最近我自己就有注意到說台股的 ASIC 也都動起來。當然前面像是創意 3443，那個算是大家都已經知道，因為它跟 Google 高度相關嘛，所以當 Google 在動、Google chain 在動起來的時候，它已經提早動起來。可是像今天我們也看到了 Alchip 也就是世芯 3661 也是直接漲停的。3661 的故事是比較綁定——應該說高度綁定 Amazon 的 Trainium，那在 Trainium 這邊它的表現是非常好啦。

只是說因為前陣子去相較 GUC 的話，我覺得 GUC 手上拿的故事可能看起來是比較香一點，因為 Google 是市場大家會去注意的一個題材；那 Amazon 的話我覺得是因為它有點像是「老瓶新酒」，畢竟 Trainium 出了這麼多代——當然它有 Graviton 或什麼。只是說我就是在過去，我甚至跟大家提過說 Amazon 的概念「估你媽再買我就一條狗」之類，當我下去我直接開始學狗叫，因為我還是買嘛——就是看到這個股價有表態，就還是會找東西去把它追回來。Amazon 相關的這個 ASIC 其實最近的表現也都是還蠻好的。所以整體來看就是大幅的資金、海量的灌進去這些 server 或者說 CPU、然後或者說 ASIC 相關的股票。

簡單來講就好像除了 GPU 比較沒有利基之外，其他的東西都會動。當然 GPU 的測試介面可能表現的還蠻不錯，但老大 NV 是停下來在那邊，然後讓其他小弟們或是各種零組件可以去做發揮。

## 聯發科：今年最大的意外

那聯發科可能算是我今年最大的一個意外啦。就是你真的沒有想過你可以在聯發科上面看到這麼多根長天，那同時當然心中就會想說「早知道我就都把它縮哈聯發科」。可是這種事情是沒有後悔藥的啦。我們在市場待久也知道說沒有必要在那邊望洋興嘆，因為總是有下一個機會，重點是要去抓下一個機會。

那聯發科一次反應到 3000 多塊去的話，不知道在我心中就會覺得說它算是非常充分的、已經先去預期後面的一些狀態了。所以呢它會變成有點類似之前的像是 NVIDIA 那樣的一個情形，就是它在上升段那邊因為直接把預期都打滿了啦，市場直接很阿莎力的去反應你的估值。所以我相信是噴到某個地方之後，可能就會要橫一陣子啦，就是要開始去驗證可能在過去的這些不管是分析師的報告或這些小作文寫的內容，是不是都有去實踐。

其實我們目前看起來，還是覺得說市場的那個預估是有一點打得太滿，包含說覺得說它可能有去做到 main die。因為在我們這邊的認知，在節目的過去數集有跟大家分享，他最主要就是在做 IO 然後跟有一個 memory buffer 的，就是 memory 相關的一些管理的這些 die 啦。所以呢我們是還沒有看到他在 main chip 有什麼東西，但是因為有一些分析師已經把他寫出來，我們就是會去尊重他。看起來市場是非常買單嘛，所以覺得說他已經是漲到一個就像是 Intel 或是這些 CPU 相關的類股。結果我自己不會想要再去加大部位啦，可能就是放著然後去觀察說到底這個市場可以瘋狂到什麼樣的一個程度。

自己在這個盤上的交易成交量呢，也算是放得還蠻大的。因為在行情好的一個狀況之下，特別是手上的部位幾乎都有拉出去的一個情形，那我們就會持續的去放大一些槓桿。那放大到什麼時候？像今天其實就是一個蠻好的降槓位置，就簡單來講就是有風吹草動，你就會去把東西稍微的調一下，然後去觀察說有沒有一些新的東西會跑出來。像今天盤中在那邊屌殺的時候，其實就注意到說有蠻多的標的在尾盤就直接拉起來，那一般來講那些在尾盤拉起來的東西，有可能就是一個小型的類股輪動已經發生了，可能就會是市場想要去關注的一個對象。所以最近的成交量是拉得比較高一點，槓桿的倍數也是拉得比較高一點。但是遇到像這類型的迴盪，像今天我自己是沒有去降槓——理論上是要去降一點，但簡單來講因為過去真的是拉太遠了，所以不會急著在第一時間就去做調整。如果後面有在一些迴盪出現的話，可能才會去執行降槓。

## 派對警察：寧願被警察帶走也不提早離場

那就像我可能之前有跟大家聊過，我現在的觀念就是：我不想要在派對結束之前就走出去。我寧願等到這個派對開到一半，然後突然間警察進來說「啊，臨檢」，然後看證件了，然後你發現說「幹，你隔壁這些老兄、這些老妹全部都沒有帶證件」，你不知道這是跟一群牛鬼蛇神混進來或什麼的，然後可能大家一起被帶回去派出所、觀察一下之後然後你才被放出來。

但不管了，所以你會去吃到一些回檔、你會吃到一些受傷，但不管，發生就是這個行情下很瘋，那就是會選擇一路搭——搭到出那種大盤格局的長黑 K、什麼吞噬之類的，那接著再看部位怎麼樣調整。但是其實老實講，按照現在這種斜率——就大盤拉的這種斜率——它就算掉下來、掉很兇也沒關係，應該很快都會再出反彈。簡單來講就是說到底發生什麼事情我們也是不知道啦，反正就是且戰且走。

那心中掌握的就是：我知道市場上的故事在哪裡。我知道現在 CPU、我知道 ASIC、我知道他們相關的一些 components、我知道被動元件，那當他們都持續的去走、我也知道它後面有 fundamental 的東西支撐的話，那其實就不會想太多。反正就是看這個行情可以走到哪裡，然後就跟著搭車就好。這是我自己在最近處理這個盤勢的一個心得。

至於像過去是可能兩三萬點、三萬點的時候有跟大家聊到說，那時候我們去看 TSMC，其實你按照它在 2027 的一個 EPS 估算的話，我們是覺得說有機會可以到四萬點。那只是說現在四萬點已經到了，那心中會不會覺得說有機會可以到五萬點？還算不到。所以自己會稍微的——也不能夠說保守，因為反正那個牌都已經推進去了嘛——反正就看會走到哪裡，然後再去識別說到底市場是願意給整體的盤面到怎樣的一個估值。

但其實我覺得市場裡面蠻多東西都是相對的啦。就像說像過去噴很兇的一些光通，最近開始出現一些回檔嘛，這些光通股其實有些是交易在兩三年後的一百倍喔。簡單來說，市場都願意去買這種東西了，所以你說市場裡面的這種大型權值股它只交易在 20 倍，甚至是這個怎麼講——就是高成長性可是都只有 30 倍左右的一些 ASIC 這種大尺寸的，那他們就顯得是非常的便宜。簡單來講，如果大家可以接受那個東西存在的話，那這些東西其實都非常便宜。那我們就是去保持耐心，當你們都可以去買那些東西的時候，那我們就可以來買這些東西。直到什麼時候？直到可能大盤走到某個地方做了個頭，那大家知道說估值的上限到這邊，那我們才會再去執行下一波的 capital rotation。

## Palantir 財報：No AI Slop Zone 與 FDE

後面我們來聊一下，因為 Palantir 他開了一個財報。那相關的財報數字，因為現在 AI 都非常方便，大家去問 AI 他就直接把數字都給你，所以我們不用去講太多量化方面的東西，我們去聊一些質化相關的內容。

我覺得他裡面講的一段深得我心，或甚至是可以直接就呼應到我們上集所說的內容，就是提到說他們現在要去搞一個類似說 No AI Slop Zone。這什麼意思呢？AI Slop 就是我們講說 AI 在網路上去產生的一些廢文、廢圖什麼的。最近有看到在亞馬遜的這個 e-book、就是電子書的產製量，最近那個數字是拉得很高。為什麼？因為可能一堆人透過 AI 然後就開始去出一些垃圾內容。當然我們不是在講說每個東西都是垃圾，可是一定有很多東西就是粗製濫造的東西，透過 AI 的協助才有辦法把這些作品生得這麼快。

那我們也是相信說當然有 AI 的協助可以讓一些作者——他可能本來就很有天分，他只是加了一個槓桿、加速器一樣，然後把這個東西可以做出來。但反正總之我們知道在網路上的內容它其實是越來越廉價的一個情形，因為產出的量實在是太大。那這我們就把它稱為是 AI Slop——不管是文字的、不管是圖片，或是一些觀念的傳達、或是一些影片。你開始滑一些短影你會看到說什麼去切一個玻璃水果然後發出奇怪的聲音或什麼的，或甚至是那種高度擬真的川普的影片或什麼的，這些東西呢都是我們可以把它放置為是 AI Slop。

當然這 AI Slop 它如果是一個像是迷因性質好笑的東西，那絕對是沒有問題嘛，反正大家看了就是笑一笑就結束。可是問題是當你今天要把這些 AI Slop 拿進去公司裡面去做應用的時候，它立刻就會產生問題。你在公司的應用你不可以去出一個模稜兩可的東西。如果你今天只是好像你的老闆——你不在一個投資公司上班，然後他可能也沒有叮嚀盯得太緊——他會想說「你去幫我去申一個 AMD 的 earnings call 報告」，那這個其實你現在真的透過 AI 就立刻做出來，老闆可能還覺得說你蠻有生產力。記得要去把那個「不是而是、不是但是」什麼這種刪掉，因為那是 AI 的用詞，這個老闆會抓到。除此之外，就是這種可能比較偏向一些概念式分析的東西，他不用抓得太準，他只要抓到一些關鍵字或什麼，這個交給 AI 做是沒有問題。

可是當今天是處理到帳戶會計、法尊人事，或者是有去對到你的客人的話，那這個東西就要求到非常的精準。所以 Palantir 他在努力的一塊，或是說他在推動的一塊，就是說這個 AI 必須要盡可能去降低所謂的 AI Slop。它的 FDE 某個程度上就是在扮演這樣子一個角色。簡單來說這個數據就是 Garbage In, Garbage Out——如果你今天是餵垃圾的話，那出來的東西也高機率是垃圾。如果說今天公司只是隨便去串一個網路上的 AI 去做生成的話，你生成的東西不只是不準，還有可能會對公司造成一些傷害。所以非常精準的 AI 是需要的。

這件事情呢，我最近在網上有看到一個說法非常的有趣——忘記是在誰的留言區然後有提到這件事情——就是說現在的網路就是大家都宣稱自己有傳說廚具，可是我們至今還沒有看到傳說料理的誕生。

什麼意思？傳說廚具就是說因為現在每個人都有 AI，所以每個人都可以宣稱說「什麼 AI 多屌、要取代誰」（最喜歡講取代別人都不會說取代自己）。那接著就說「我今天用我 AI 又刻出了什麼產品、我今天用我的 AI 又怎麼樣」。那我就問你各位，那你他媽產品到底在哪裡？所以其實現在很多人在用 AI 做的東西，他可能自己覺得說「這是我一個小創作，我可以做出一個我以前沒有辦法想到的應用」或什麼的——你也看到很多人開課跑出來講這個「我用 AI 生成了一個什麼小 gadget、一個小功能、小 tools，這個東西非常好用、以前我可能要買或什麼的」——但這些東西已經是非常嚴重的一個泛濫，就是在市場裡面是膨脹到基本上猴子都可以做出來。所以你現在可以做出這個東西一點都不稀罕。

那最直接的驗證就是：你一定沒有辦法拿這個東西去賣錢。你會看到很多小故事告訴你說他做一個什麼小應用賺多少錢或什麼的，但你會發現他是鳳毛麟角，因為像他在做這件事情的人實在是讓他太多了，可是真的有辦法把他變成是可以營運下去、可以去獲利的一個產品的人，這很少的。所以產品本身超級重要。

## 軟體股「無限重要」：Stand your ground

這個 AI Slop 的觀念就會反映在未來的一段時間，包含說像我們上集有稍微的提一下 NICN（譯按：應為某 SaaS 公司），即便這家公司並不是自己有在觀察或是有布局的東西，可是它開出來的數字是顛覆市場想像的。因為市場本來認為說你就是死定了。

那特別是要知道說這個東西死定，它不會是一個未來，它應該是現在進行式——為什麼？因為過去你已經看到巨頭或是一些 enterprise，它確實是 shift 一大堆可能過往的 IT 資源、或者說會去買一些軟體、買一些有的沒有東西的資源，全部拿去買 AI 伺服器，這個是已經在發生的事情。所以一定是有人受害的——就是你的錢不是說會無限增生的，當你拿這麼多的錢去移到某個地方一定有些地方錢會不見。所以如果今天你注意到有些公司他還是有錢進來，那他真的是有機會可能就是錯殺。

因為在過去市場給大家的一個論述就是——即便老黃、就是 AI 教主他自己都反對，他說應該是要創造更多工作、軟體會變得比既往更重要或什麼的——但蠻多人的想法就是說這個以後都不需要有任何的軟體。其實這個是很矛盾的，因為如果是這樣的話，那科技巨頭應該全部都要丟掉。不是說不用軟體，可是一定會有一些可能典範轉移的事情發生，或是一定會有一些汰弱留強的狀態發生。

那這個也不要講說是軟體，其實你說硬體也是嘛。這陣子如果你在做工業電腦的這些廠商其實就超級危險。為什麼？因為簡單說就你的那個採購議價能力本身也沒有人家強嘛，你料也拿不贏人家嘛，所以那些五哥十哥他去做 AI 伺服器，即便那個毛利數字沒有這麼高，可是他至少可以確保就一直賺錢嘛。那你那種採購力沒人家強的，你現在又要去買那種貴森森的記憶體，然後板子又變貴什麼都變貴，所以一樣是在做 server，可是其實有些人的財務狀況在後面是有機會出現爆雷的、出事情的。

所以不是說每個做硬體的都一定會好，那一樣的道理就是不是說每個做軟體的都會好。但是你已經開始注意到——特別是我覺得像我們節目去提到自己喜歡的軟體（這當然是不知道自吹啦，但你回頭看一下）——像我們特別喜歡的 CrowdStrike、或是 Palantir、或是這個叫什麼，Cloudflare，他們在過去一陣子其實 year-to-date 有兩家是正的，只是 Palantir 還是負的還是在水下，但相較於一些 SaaS 公司可能是已經跌掉 40%、50%、甚至是有跌掉 60%、70% 的，那他們是相對撐得住的。

所以我覺得這裡面要傳達的一個概念就是：你沒有辦法很籠統的去聽市場的某一種說法，然後就把投資這件事情做好。你要有自己的一些判斷，那這個判斷不代表你一定是會對——就是一定會有錯的時候產生——只是至少你要 stand your ground，就是你不要說今天聽到一些風吹草動然後立刻就改一個說法，因為這對自己的 credit 也很傷。

那我知道絕大多數的散戶你不用管 credit，可是在我們自己的工作領域裡面，因為大家其實是會互相推標的——就是我跟你認識，雖然我不是投資公司，可是我們互相「你在看什麼？你曾經都推垃圾」，久而久之大家知道說你就是一個垃圾。所以你不能夠就整天改說法，然後或者說人家每次問你標的，你就是「手上都是強勢股」聽起來好像很屌——很多散戶看到那種有些人手上都是強勢股覺得說這一定是神的——沒有，這些東西你只要每天打開 XQ 你就那個漲幅排行榜，去橫盤的話才會有機會會去找那些東西。

## 一軍 vs 二軍：環球晶 + 世界先進候補

就有點類似說其實我現在手上看得到的一些劇本，你說像那個矽晶圓——環球晶雖然昨天講的不是特別好，但整體來看就是矽晶圓的一個週期應該開始要上去了。那這就不會是首選，他是排在後面的一個選擇，雖然他可能最近股價也蠻強的。

再來另外一個我覺得也蠻意外的是像這個 Vanguard 就是世界先進。世界先進他在最近的一次對外的說法也是顯示了他一些廠房他是在做那個成熟製程的，就成熟製程的廠房他可能還沒有蓋好然後東西都賣掉了，他的利用率也是超級高的。那雖然這個其實大概在幾個月前就看得到，但他就是不會是一個首選。

所以我們現在的概念有點像是說我們自己的心目中會有一些一軍跟二軍。一軍就是股價確實在右上角、然後故事又對的東西。當一軍今天開始有一些軟掉的情形產生的時候，那就會選擇可能部分往二軍去；或是說可能一軍拉回夠深的話，有機會再去做更進一步的布局。但整體看會認為說以下這個盤面，就是它那個斜率真的實在太高了，所以一般這麼高的斜率不知道——除非是像那種打抗炮那種就後來就直接 A 下去之類的，不然很難啦——就是一般一定會有一個後面會給一個盤整段，那盤整段就會去拼選股，到時候就可以在這些東西去做一些輪動。所以自己心中的規劃是這樣。

所以美股的一些在過往可能是所謂的被 AI 排擠、被殺到的東西，一些 SaaS 或者說 SaaS related 的東西，它只要是財務數字夠好的，可能後面都會想去注意它，因為他真的都蠻多跌到所謂的便宜的位置。當然便宜有可能是一個價值陷阱，只是如果有財務數字可以去做背書，那如果說公司給的 guidance 也是夠高的話，有可能後面會產生一些市場論述的翻轉現象。就像說在 CPU 這波轉好之前，所以我覺得市場裡面有很好的記憶力，因為你現在只記得大家都講 CPU 很好，可是其實在幾個月前是不會有人跟你說 CPU 很好的，甚至你問大家、大家講說「誰會去買 CPU？」。有時候就是在一個瞬間，這些東西就會去做翻轉。類似說可能在今年年初的時候，有些人就直接跟你講說「啊，美股死定了，對不對？說不要再做美股了什麼的」——surprise mother fucker，直接就一路噴。

### **魯米**：跟另一半聊天的平衡

> 孟恭去嘎你好。請問孟恭會常常跟 Lisa 聊天嗎？我的另一半說我跟他聊天都聊太淺，但我的工作跟他相反，是需要接觸很多人群講話、很耗心神氣神，那伴侶說他需要花很大的努力我才會想將話題往下帶入分析。想請孟恭開示怎麼取得之間的平衡。謝謝您，祝全家健康平安順心。

所以兄弟完全就是問錯人了啦。因為以小弟的狀況來講，我老婆也是很常說「我怎麼都不講話？你在節目不是很會講嗎，你平常怎麼不講話？」。

那我覺得也不是因為我節目把話都講完，是我平常真的話越來越少啦。我覺得我以前話很多，可能某個程度上是我是一個 people pleaser——我就想讓大家開心，我就覺得說不知道大家尷尬在那邊就很尷尬，所以就要講一些笑話、就要去做一些事情或什麼。但我覺得他是完全可以接受我就是最安靜的那一個人。所以我在家裡我也是不太會去講東西。

我唯一比較熱衷的時候是講我老婆聽不懂的東西，而且我可以看到他眼中有一絲的放空，可是還是盡量的去回應我。像我跟他說我一些研究發現或什麼的，他不是特別在意，然後他就還是會聽我講。那這時候他就會抱怨說「你看每次我都會聽你講話，可是我在講話的時候你都不聽」。那我都會說「不是啊，你跟我講卡戴珊又怎麼樣，你都是跟我講一些花邊新聞，這些花邊新聞我就真的是 give zero shit 嘛」。那他說「老實說你的工作我也是 give zero shit 啦」。那這時候我就會說「沒有啊，我的工作是不知道有生產力的啊，蠻酷的啊」是產業——各位自己講講我也知道說，對啦某個程度上你知道他那些花邊新聞他也是可以有生產力的啊，對吧，他也是可以——你知道有些那種娛樂圈的人就靠這個賺錢嘛，所以他跟生產力與否是沒有什麼太大關係。就只是你到底有沒有用心去聽別人講話。

所以我後來回去訓練，就是當我太太在講話的時候我盡量不要讓他講兩到三次。因為有時候他真的會講兩次到三次很誇張，就他已經整個故事講完——所以我說我太太是非常有耐心的——他已經講了 30 秒然後我真的是突然回神說「Sorry 你剛剛在講什麼？」然後他再講一次，我再「Sorry 你剛剛在講什麼？」我是真的會連續去忽略。那這不是一個好事情，這是非常壞的一個示範。

但我覺得近年我有開始慢慢的去改了。其中一個非常有效的做法是把自己的時間線切開來，那這個也是要非常感恩自己有孩子之後，孩子帶給我的一些改變。

什麼叫把時間線切開來呢？因為像我的工作是屬於基本上我非常的有彈性，所以我可能可以明天就買機票去哪裡，就像可能各位聽眾看到，有時候我就什麼我突然間就回娘家就兩個月，或是說什麼突然間就去峇里島或什麼的。我可以隨時跑來跑去，可是呢交換的是怎麼樣？我可能到深夜都在工作。就是我雖然是有彈性的，可是我的工作量是非常大的。那我可能今天就算是跑去峇里島，然後大家在 Villa 裡面游泳或什麼的，我可能還是坐在旁邊去敲我的筆電，去看一些東西或什麼的。那這個就會變成說你看起來是時間彈性跟自由沒錯，可是呢你會在你可能該放鬆的時候你都還是在想事情。

那後來怎麼樣去做一個切割？就像我就知道說我現在晚上的某個時間點我就一定要念故事給我孩子聽，那一樣的道理就是我可能過了晚上六點之後——後來才發現說一些大老闆他們可能在一些訪問都會提到這樣子的東西什麼「五點半回家之後我就是不接公司的電話」什麼的——那我覺得就是這個概念，就是你要去做一個切割，這個時間就是 for 家人。那 for 家人不一定是說你要侷限於自己「我就是要聽我老婆講話」沒有，你可能是 for 你的孩子，那你老婆就一起或什麼的。但至少你知道說你有時間你是空下來的，今天如果有人密你你會知道說「這個時間是神聖的我沒有辦法回你」。我覺得這個後來幫助我很大。

也因為這樣子我會知道說那他來跟我講話的時候，那這個時間點我不應該去想我自己的事情。即便我覺得我事情很重要，但他的事情也很重要。所以我認為那是一個心態上要去改變的。他也不是說什麼你的工作特別忙碌什麼，因為像我們的工作也很忙碌，那都是某個程度上的一個藉口，是說你的工作可能是已經是高度滲透到你的生活了，導致你的生活跟工作其實是已經失去平衡了。那不是說一定要 work-life balance，因為我們都知道說如果你要追求極致卓越的話你不會有 work-life balance，不會有這種東西。但至少那個平衡盤要稍微把它喬回來一點點，不然可能會妻離子散吧，這個要小心。

### **矽谷金城武**：結婚 8 周年 + 安東尼·波登

> 孟孟大，我的夢想是有一天可以像孟恭一樣過上真正自由自在的生活，想做什麼就做什麼，不用再受朝九晚五的工作束縛。而且能夠至少每一季都帶著全家人出國玩。另外想請孟恭祝我和太太——加州矽谷的 Andrea，是念 Andrea 還是 Andrea？如果念錯不好意思——6 月 3 日結婚 8 周年快樂。一路走來辛苦老婆一肩的陪伴跟照顧，讓我兩個寶貝兒子都能夠快樂長大。也祝福他早日恢復窈窕好身材，一起迎接更美好的未來。

非常正向的一個留言，看到這種東西心情就很好。

那就像我剛剛前面所提到的，對，我的時間是彈性的，但某個程度上我的時間也算是被綁死的，因為據說來說我就是每天⋯⋯像我前面甚至可以直接在某一集的開頭然後跟大家很高興的分享說，我因為網球破除了我早上一定要坐在電腦前面的這個 routine，已經不知道他媽幾年了。那我因為打網球去改變這件事情，就是心中還是會覺得有一點罪惡感，但當下是非常開心的。

所以呢，今天來講就是我們很常去看別人生活的時候，這也不是只有各位，就像我自己也會這樣，看一些大哥的生活——因為我很羨慕他這麼有錢或什麼的——但你可能不知道就是別人辛苦的點在哪。當然不是跟大家賣說我很辛苦，什麼意思就是說，你盡量不要去羨慕任何人，然後你要去小心你去許得任何的願望。你想要成為他或什麼，可是你真的去變成他、或像是我某個成上我變成了我小時候可能想要成為的某一個樣子，可是呢其實跟自己想的呢是有一點不一樣的，你還是會有你的憂慮、還是會有你的難過、有你的擔心或什麼的。

那這個就是我們很常舉的一個老題目，就是那個安東尼·波登（Anthony Bourdain）。大家一定覺得說他是全世界最好的工作、最好的生活，可是呢這個傢伙可能最後面心理上的某些關卡還是不一定過得去。或甚至是一些 rock star 也是——欸你是萬人膜拜的對象，你為什麼會選擇去結束自己的生命？

所以我覺得在生命的追求之中，去讓自己在當下就獲得快樂是很重要的。所謂的當下獲得快樂就是說你要去欣賞你生活中的點點滴滴，而不是整天就是「我希望變成誰、我希望可以怎麼樣」。你可以有一個目標，可是其實生活的快樂真的是在當下。這可能聽起來很幹話，可是好難去形容。

有點類似說我上集在聊那個酒的時候就有提到說，那個酒商老闆講說可能最好喝的酒是跟一群狐朋狗友然後他媽的在路邊喝的啤酒，你當下沒有好好去感知他是因為你可能不夠成熟。可是等到你真的可以去 appreciate 那個 moment 的時候，你已經是回不去了。

那或甚至最近看到明道的一個影片，反正他在那邊講說他錢都不會留給孩子。這個我是不認同——就是我覺得留東西給孩子沒有什麼不好，不用去排擠小孩中——但他講的觀念是對的，他就說他覺得他人生最快樂的時候是從沒有錢然後開始到有錢的那個階段那最快樂。這真的是還蠻快樂，這一段時間很棒。那你真的開始獲得一些資源，你開始不用去在這個全聯去看價格的時候呢，你不一定會是最快樂的時候。所以有可能你現在就在最快樂的一個時候、你現在就是最自在的時候，那個要體會一下了。

### **看樣子應該會在國外遇見吧**：短變長 + 適應金額放大

> 不用躲在家裡哪裡都不去。請問孟恭，以前動能也是短週期吧？關於你從很瘦小到現在壯壯的經驗分享：（一）如何改變「由短作長」？（二）怎麼適應一開始回測的金額跟後來回測的金額？雖然知道是看 % 數，但也差太多了。而年前你進市場的時候，普遍可能是噴到 3 到 5 根就沒了，那怎麼駕馭現在的超緊噴行情？還是你本身就很會 BBQ？或許如你所說的特質「你很有彈性，慣有邪笑」。謝謝抽到。

OK，怎麼樣從短變長？我注意到身邊每個短變長都是資金變大，資金變大自然就會變長。那當然你說是資金變大是本質還是說是年紀，因為在資金變大過程中大家也都變老了，你會注意到大家變老、大家都不會想要整天坐在電腦前面盯盤。所以我覺得由短變長是勢必會發生的事情，幾乎我身邊的所有人都這樣，即便是一些做量化交易的人他們可能也開始有一些長頭的部位。所以我認為那是必然的，甚至你不用去刻意訓練，就是你時間到就是會發生——類似說時間到你媽早上就是會翹不起來之類的。

再來講說這個金額。這個金額確實要花時間去熟悉一下，它不容易。你會看到你有一些回檔出現，然後接著你就很焦慮會很受不了。但如果你用 % 數看，你會發現這個 % 數可能跟之前是差不多。所以我覺得它有點像說你在短期內報酬太快、你的知識認知感受度沒有跟上的話，你會很容易去把你現在要賺到的東西丟回去——你會賠回去。那沒有辦法，那是因為你的認知是還沒有跟上、你的感受度還沒有跟上。但是一段時間之後一定都會慢慢的適應。

這個真的是有點類似說要幫自己 fool-proof 一樣，你就要知道說你不要去盯那個金額，你就是要看百分比，或者說甚至你不用整天去算這有的沒有的，你就直接用鏡子看。所以我自己是蠻推薦都用鏡子去追蹤你的部位就好了。這個是很簡單，因為你拉長一看就會發現說你現在很焦慮，可是你鏡子明明整個長線是走得好好的，那或許沒有出什麼樣的一個大問題。所以這個真的是心態上要調整的東西。

然後再講說要去怎麼樣處理下超緊噴行情。簡單來說你沒有辦法什麼東西都爆到。像我前面才剛剛舉過一些薪炊（譯按：新貴或新興股）的例子，像那個正美——正美買超不及便宜的——或是說三大紙也買超級便宜的，可是後來整個噴出去的時候你那個部位一定都是慢慢的降低。可是有些部位呢你從頭到尾都在那幹大腸的，可是呢他就是不會噴。這個勢必會發生。

但後來會知道說當你的部位變多變散的時候，你比較容易吃滿波段。可是當你想要去把部位都壓滿的時候，你勢必會在漲多的時候你就去思考說「是不是我要賣一點？」。對你看起來短線上是賣對了，可是很多時候你長線看會發現「早知道我都不要賣」。但是你也不能夠這樣去把眼光都盯著同一個東西，因為你換來換去說不定你最後面換到一個你最後的真命天女，然後幫你賺超多錢之類的。所以我認為這個也是一個過度被聚焦、然後可能不是特別重要的東西。

只要你的鏡子是一直有在上去就好。只要你跟大盤比你有 alpha 就沒有什麼太大的問題，那過程中的一些有的沒有的事情你可以把它視為是某一種 collateral damage，大概這樣的一個感受。
