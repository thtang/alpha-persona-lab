### AWS Chain 的表現

那本來我自己的假想是說，在 AWS re:Invent 這個活動辦下去之後，可能 AWS chain 會有一個發揮的機會，就像前面 Google chain 是表現得非常好。那最後面就整個說，可能我自己烏鴉嘴，因為我已經一直長期跟大家講說 AWS chain 不是人玩的地方，但是我自己其實本身還是持有蠻多 AWS chain。

雖然現在是以 Google 的持股作為一個主力核心，可是 AWS 的持股不管是什麼板材啊、交換器啊，然後或者是它的什麼 Scale Up、Scale Out，其實都是自己的一個選股之一，甚至連它的後段的 IC Design Partner，就是也是有在自己的配置裡面。

那看得出來是兩樣情啊，就是在 Google 的相關的活動跟曝光之下，它的供應鏈是有上去，可是 AWS 的活動之後呢，它的供應鏈是全部下去。

### NVLink 採用的解讀

那可能是因為市場過度去反應它要去採用 NVLink。那這個 NVLink 呢，它其實並不是說就是它要放棄自己的 NeuralLink，然後全部都要走向 NV 的一個解方，而是呢，它就是要去把它的東西長在 NV 上面，你可以這樣去想像。所以可以把它採購的 NV 伺服器跟本家的 XPU，也就是我們講 ASIC，可以去做一個整合。

那就我們的理解，目前它的路線選擇上是一個雙軌的設計，也就是它會有這個 NVLink Fusion 的一個做法，那同時也會有它自己本家的 PCIe 的一個做法。

所以其實理論上看起來的受害者會是 Broadcom 的 SUE，或者說前面剛跟大家宣布說要成立的這個新聯盟，可能是在這一波裡面沒有拿下太多的優勢。可是本來既定的 AWS chain 的一些交換器或是相關的廠商，以及像是 NVIDIA，對我來說，他們其實整體來說都是受惠的一個狀態，包含像是 AEC，也是一個受惠的狀態。

所以不太懂，就是這個市場為什麼會這樣子去反應，就是在這個事件之後，然後所有 AWS chain 都直接往下跳，這個我們不太知道。可是有時候，其實在短期的股價上，那是有很多你猜不準的事情，你以為會這樣就不是這樣。

那可是像 Credo 前面也是有重殺過嘛，就那種你不知道為什麼會這樣子跌，那時候在節目裡面一直跟大家抱怨說，我真的不知道為什麼他會這樣子跌，那我們台灣的貿聯在那邊創新高，然後他在那邊跌。那結果呢，財報開出來你就發現說，就美國人真的比較慢嘛，然後他們可能沒有意識到這件事情，所以才會在就是盤後直接噴拉嘛，然後後來在開盤的時候我曾經最高是跳到了大概 23、24% 左右，只是最後面是直接收斂回 10% 啦。

那像這種地方對我來講就是一個殺人 K。我的反應是比較慢一點，一般我可能就是說，我發現你的那個圖越漲越母湯，然後又爆大量的時候，我會去做一些減碼。那這一波呢，因為剛好身邊有一些這個技術猛男，他也跟我在同一個車上，他直接跟我講說這種開高、然後一次衝那麼高的，然後同時你已經看到說台灣的供應鏈在同樣的這個財報——因為是 Credo 的盤後開嘛，那開完之後我們看貿聯是直接開高走低就灌下去了嘛。那他的意思是講說，基本上可能玩家都是同一批人，所以貿聯會這樣子走的話，那可能在 Credo 這邊也會這樣子走。

我真的覺得就是這位技術猛男超級厲害，就這邊就被他講中。他是賣在最高點，那我是沒有賣在最高點，可是我也是沿路跟著丟，那所以就稍微的去做一些這樣子的調節。就最近蠻多這種，就是可能你過去在抄底的東西，然後現在要彈起來，你發現他力量不夠，或是說表現得不夠好，然後就開始去做一些減碼。

也稀微跟大家分享一下，就這次我看到 AWS 的這個活動我的心得，就是我認為說市場是有一點，甚至可以講說是反應錯方向了，我認為是這樣子。當然那個短期的股價就是長這樣不好看，所以不知道，有可能需要整理一段時間再看怎麼樣。

這個市場是大到就是可以讓這麼多玩家都在裡面可以賺到錢，就即便你現在看可能有很多的參與者、有很多的 chipmaker，或是說相關的電纜的廠商，可是其實大家都還是有辦法分。它跟散熱的環節有一點不太一樣，就是它的那個長長線其實超好的，那可能是因為短期過於的擁擠，所以會有這樣的一個變化，目前我會這樣子去做一個結論。

## 散熱供應鏈觀察

那跟大家前面聊到這個散熱這一邊，我們過去幾集有跟大家聊到一些散熱相關的話題，那現在有一些比較明朗的狀況。

### CoWoS 降價

那首先在 CoWoS 這邊，我們是知道說 Group A 的一些 Components 其實都有遇到一些殺價的問題，目前是知道說有些朋友是喊到比較誇張的數字，說有達到六折左右，那只是我們目前了解到大概是七八折的狀況是有的，也就是整體會去做降價。

### QD 供應商競爭

然後再來就在 QD 這邊，Supplier 是跑到了五十幾家去了，所以這個就是用屁股想都知道說一定會出現一些降價的狀態。

那當然其實像這種像是 CoWoS 或是說 QD 的一個降價，它不一定完全是一個很壞的消息，因為是整個市場它還是在一個擴張的狀態嘛，所以你可能就是說你的 P 是下去了一點，可是你的 Q 是大量的往上拉，那 P 乘 Q 之後，欸搞不好還是不錯的。但我相信那個估值乘數就是相當程度可能會受到一些影響。

### Kinsly 的狀況

那前面提到的像這個 Kinsly，那他也是有遇到一些降價的狀態。就是過去可能是 NV 故意不把它放在 AVL 裡面，希望它的專利方面去做一些讓步。那我們現在瞭解到就是應該是不會在這上面去做讓步，反而在價格這邊就是可能跟你稍微的談一下，然後那還是在接下來的 AVL 裡面有望會出現。那所以接下來我們可以稀微注意一下這一塊。

### 散熱供應鏈總結

這個在我們過去幾集其實有跟大家聊到，現在越來越明朗，就是在一些散熱的供應鏈上面有出現了可能被要求要打折的一個狀態。那我們希望這個打折，可以被伺服器接下來出貨的數量，或是說像 Google chain，它假設有 upside 的話，那可能整體可以去 offset 掉這一些降價的壓力，那就是會繼續走得不錯。只是如果沒有的話，那可能這一部分要稀微的停看聽一下，就稀微注意一下。

可是就整體來看，我還是回到我的大原則，就是我認為說這些東西都是很明顯在一個趨勢的跑道上面，所以理論上應該是就是假設有人犯錯、有人要重殺亂殺，然後接著跟你講說這個東西以後不行了或什麼的，其實反而是我自己會想要介入的一個時間點。

### 操作策略轉變

### BCI 上漲的原因

那當然它有幾個故事在背後啦，包含說像在西非這邊可能有新的這個鐵礦的產能開出來，然後再來呢，現在市場上有一個 rumor，當然這可能傳很久了，就在講說烏俄戰爭有可能會有停戰的可能性。

那之所以會想分享這個是因為，剛好昨天在跟老婆聊天，然後就要聊自己的工作。平常跟她聊工作可能她都不一定聽得懂，要跟她聊什麼電子零組件、要跟她聊伺服器，她一定是完全是聽不懂，晶片聽不懂。

### 現貨船 vs 長約船

然後再來就是也跟她細部的拆解了一些小東西。但這個本身並不是可能我們在股市裡面最研究的一個主題，可是有時候覺得有趣你會去把它挖來看。

那像今天說如果接下來我們注意到 BCI 的一個價格持續上漲的話，那請問一下你要注意的東西是什麼？BCI 的話就是在講這個 Capesize 的船，當然你就要去注意說這種海峽型的船誰是最多的。那就算你可以找到誰是最多的，你還有進一步要去推導的就是，這裡面有些是簽長約的，就是我們知道說有些可能是為了獲利的保障，它不想去扛各個景氣的循環或什麼的，所以它可能是用固定的某個數字、某個毛利率去跟人家簽一個長約。在這個約定的時間內，反正對方就是穩穩的付租金，所以如果這個報價突然間上漲，跟這種長約的一個 supplier 可能就不會有這麼大的一個關聯性跟影響。

那有些呢它可能是走現貨船的方式，像大家說美國有一家叫 SBLK 的，它就是幾乎都在走現貨船。那台灣的話，像裕民它也是有在走現貨船。那現貨船的話可能就會隨著這個報價的租金去往上去做調整，那它可能就比較容易吃到這樣的一個行情。那當然反之，如果今天報價的行情不好的話，那那些打長約的可能就會有比較好的一個機會。

### 投資邏輯的相通性

所以其實整體看起來，你會發現說你是一路通路路通，你沒有辦法可能在每個行業裡面變成是那種就是真的專家學者、超級厲害的業內人員，可是本質上是差不多的。

像半導體也會有些人，他是選擇是用比較這種短期的報價去做生意。今天如果說報價是漲到一個比較高的價格，那可能很多人都想要簽長約，像以前的晶圓代工廠就是因為這樣去簽長約，簽長約就希望可以把這個報價鎖在這個位置。或是類似我們最近看到像三星這邊就有在講說，它的這個手機部門想要去跟記憶體部門去要東西，然後希望可以去簽一個約，那對方就不願意簽約。它意思就講說，因為這個報價還會持續上漲，所以我當然不要跟你簽，因為我可以在這個報價的循環裡面獲得很多利益的話，為什麼要把報價鎖在這個地方。

所以其實那個觀念是通的。它只是說，因為我們今天用半導體用手機去講，她可能不一定聽得懂，可是用這個海運剛剛去形容的話，她就可以完全的了解。

### 投資研究流程

所以其實就是說，像以這個案例的話我會怎麼做呢？就是我知道說它如果有一個最大的利多，會是在烏俄戰爭的一個停戰嘛，所以我會先把這些東西可能都放進去追蹤裡面，然後再就是按照上面的一個挑選嘛。

就是說航運的指數有這麼多種，你要挑哪一個？那我們要選擇是選擇散裝輪的一個報價。那散裝輪裡面又有分什麼 BDI、BCI，那你要哪一個？你選擇用 BCI。那 BCI 的話你就要去聯想說，那誰這個海峽型的船是最多的？然後誰的現貨船是最多的？它就會直接受到這個報價的影響。

那當把這個東西都已經篩選出來之後呢，我們就把它放在追蹤清單裡面。那當然你有幾個做法，一個就是像現在這些市場的猥瑣之輩一樣，就先進去蹲，或者說你就先去搶快。那可是這個搶快，這個都是賭的成分是比較大一點。

那也稀微把這個思路跟大家分享一下，就是在觀察市場的一些股價的變化，然後找到一些機會，那因為找到這些機會，後續怎麼樣處理的一個流程，我都跟大家分享一下。那像這個就只能夠算是我們可能一整天在做的事情的其中一部分。
