接下來我們進入市場話題。

我是覺得川普應該是會選上，在這次的辯論會之後，我覺得很明顯地看得出來，不管是民調或是一些 KOL 的意見，像是美國的一些政治評論相關、或是一些網路上的意見，甚至是一些傳統的新聞台，以前可能都是全部站隊拜登的，但發現拜登這次表現不好，你就注意到這些高級知識份子很多就開始切割，他不是我們的，別人可以把他撿走。那拜登在自己的社群媒體是表示說他會選到底，只是在我來看，當然我可能不是政治的專業，但我的意見是，我感覺川普是會直接選上的。

川普選上的話，當然會對股票市場會有一些影響，大方向格局我覺得不會改變，但是一些細部的影響，我們可以稍微來討論一下。其實對我自己的持股來講，我覺得影響也是不大，但如果你今天手上持有的是一些能源股、新能源的話，我認為這個是會受到影響的項目。那如果你是持有一些所謂的傳統能源、頁岩油或是石油的話，可能會有一種轉機出現的可能性。但我們講的都是題材面相關的東西，所以題材面就是說，當你今天知道川普可能贏面越來越大，或是確定要贏的時候，我覺得這東西可能會動一波，但是真的上任之後，不代表這個東西就會一直往上噴，因為最後面還是要回歸世界運作的一個本質。

就好像之前選舉的時候，很多人講說誰上，然後這個股票就一定會怎麼樣，其實都是認知可能有一些偏差。整體上來看，我覺得當然政治人物對市場的影響絕對是大的，但不是完完全全都是因為政治人物，然後去導致市場會往某一個方向去走。這也是很多可能小白搞不清楚的，所以他們很多時候會用股票市場的走勢，然後去解釋說某一個新聞正在發生的一些政治事件，這其實超白癡的，有時候講完隔天就會被打臉，那他們還是樂此不疲。那我們就不去評論這樣子的人，但我們要知道說，我們今天要看一個大格局的東西。就是我覺得，如果你相信這個國家，或是你相信美國，或不管台灣也是，你相信這個國家的整體表現是好的，他不會因為一個人就可以把這個國家給毀掉。很多時候就算可能看起來有一個戰犯，可是那個是連續的錯誤導致整體的總經面是不好的，但不太會是因為某一個單一的事件。

所以當我們今天去看美國選舉的時候，我們不會去評論可能誰上，這個股票市場就會怎麼樣，沒有這種事情。我覺得就是題材面可能有一波，那在實際運作上的話，我不會覺得因為可能川普上，所以新能源就要全部完蛋。可能有些補助會被砍掉，有些政府的採購會被砍掉，可是我認為如果這些新能源，他是真的有搞頭，而不是完全就是吃補助起來的話，那他會起來就還是會起來。

覺得說一個很大的爭點，就是說可能川普上任之後搞不好電動車就會猝死，這個我覺得不會。因為我認為電動車是一個正確的趨勢，可能在川普上任之後，一些補助會被砍掉，但電動車的滲透率持續加大，我覺得這是不會改變的事情。但是這個補助被砍掉，有些人就受到影響了，所以我們要討論這個。或是一些新能源的採用、或是一些限制，可能會要求說我們在什麼 2030、2035、2040，能源要怎麼樣改。這些東西如果說沒有像拜登這種左派政府去強制要求說什麼時候一定要怎麼樣的話，那可能這個格局就會有一些初步的改變，這可以稍微注意一下。

但還是要回歸到我覺得大重點是，我還是相信新能源的一些發展，RE100 等等的，所以我認為他們不會有那種毀滅性的一個結局產生。但在短期內，可能會有一些消息的刺激，會對股價造成一些影響。所以結論就是我認為，這只是一個題材相關的東西，大家不用太擔心，它可能是一個短中期的事件型影響。那就算他可能真的上台，對電動車做了一些什麼樣的表態，也不用太擔心。因為像已經注意到馬斯克，他有大量地跟川普政府在疏通，有去做了蠻多的表態。那其實他非常賭爛拜登跟 Kamala Harris，這邊應該有在追他 X 的都知道。所以我會認為說，不是說什麼川普上，就非黑即白，他一定會把電動車弄死；也不是說拜登上，那新能源就一定會飛天。我覺得很多時候也知道要回歸一般的商業邏輯，就是這東西到底本身有沒有搞頭。

所以我會覺得像是電動車或是說 Tesla 的影響，應該都是有限的。但是我們不排除，短期的事件上可能有一些股價衝擊，就是市場的羊群們情緒的影響，有可能會出現，那對我來講是小事情。

不過有一個我們可能會稍微在意的點是，但這我們也只能夠用猜測的，就是目前在供應鏈上其實是有 check 到說，像之前有跟大家提到，越南有可能是下一個會被制裁的地方。那再來就是在關稅的部分，有可能再往上拉。如果是川普上的話，我們是猜測說他對中國的一些封殺可能會拉得更高，甚至有可能會透過去針對中國的一些封殺，然後來逼中國可能把俄羅斯壓上談判桌，直接去結束烏俄戰爭，去達成川普所提到的「我可以結束烏俄戰爭」。很多人覺得是嘴砲，他搞不好用這種方式去做，這個都是我們有在猜想的事情。

但這其實比較偏向飯後聊天的一個話題，就是我們都知道國際政治有太多的變形跟變數在裡面，它是大國之間的一個博弈。所以對我們來講，我們會去討論到這樣的東西，但這其實對我們的操作不有太大的影響。但真正有影響的會是關稅這一條。

那我覺得，but then again，我們都要明白，不管今天是川普上還是拜登上，對中的一個強硬態度，我們都相信會維持，這應該是不會改變的，因為有商務部這個單位的存在。但是如果今天是拜登上的話，可能就維持既有的路線，但川普上的話，可能會去加強，可能會去加一些猛藥。那如果說真的有發生的話，那可能就會是蠻有趣的。

但是我們可能找到一些潛在的機會。那當時其實我們雖然知道說 AMD 跟 Intel，它跟 NVIDIA 一定是有一些差距，因為 NVIDIA 在軟體的根基上得非常強大，它 CUDA 是一個很厲害的東西。只是我們當時其實沒有辦法知道，差距原來這麼大。那是後面可能等到 2023 下半才開始越看越明顯，甚至是直接發現說那個距離超遠。然後到現在我的結論是一句話，基本上等到這個 B300、B400 以後，我們再來看它，前面的東西就不用看到了，那個 B300 之後可能是一個比較關鍵的一個節點。

那我就把這個思維帶進去軟體裡面，我最近就想通一些事情。因為我在美國有看一些軟體，但美國跟台股我都有去找一些小型的軟體去做布局。那美國這邊就比較好直接講，因為台股會被上一些什麼影響市場的說法，所以我也不需要影響市場就可以賺錢，但是有些人就會拿這個套在我頭上，所以我就選擇不講。那這不是我故意在那邊裝神弄鬼，有一天日子可以跟大家開牌的時候，到我感覺夠舒服的時候可以講。那我已經可能過去的紀錄都已經可以證明說，有沒有講，反正都是一樣會賺錢的時候，那那時候再來跟大家講，反正也是慢慢在累積中。

那像美國我注意到自己一個不錯的小型的軟體公司：Palantir，這個是 2020、2021 我們就會聊到的一家公司。當時可能比較著重在講他國防相關的業務嘛，現在比較著重會是他的 Ontology 跟 AIP 相關的服務。另外一個 Confluent，他做的就是 Apache Kafka，這個東西是一個開源式的內容。很多人覺得因為是開源式，所以可能會被取代，但是我不這樣認為，我覺得資訊流在後面會是一個很重要的重點。

所以我就突然想通一件事情。我最早跟大家聊軟體的時候，大概是一兩個月前，那時候我一個說法很北爛，可是其實是當時我自己的想法。那我也因為這樣子，有去壓這些所謂的代理商，就是我認為其實市場一開始會看不懂，所以他們會亂炒，然後就會跑去拉代理商。因為代理商很明顯，誰代理 Microsoft 的產品，就他，然後就買他嘛。這個對市場來講，就是一個很簡單的邏輯。所以我算是去賭這個事件，我知道說大家會無腦亂買，所以先去買，好像也覺得還蠻爽的。

但我現在認為說，那時候我壓注的理由是「大家會亂買」，可是這樣我覺得好像歪打正著，就是這種代理商可能在後面會產生一些特別的機會。為什麼？因為既然我們今天沒有辦法知道軟體的贏家是誰，可是這些軟體他們目前在我們現實上看到的一個特色就是，你要把 GPT 導入你的公司裡面，你如果不想要停留在它只是一個比較聰明的對話機器人的話，你要讓它可以真的去為消費者解決問題，可以針對消費者不同的需求，以及去針對公司所可以提供的服務，把所有的東西都綜合起來的話，你必須要去 fine tune，或是你要去把你的資料變成是向量的資料，要去規模化，這樣子你才可以讓這些資料真正的被取用。

而這些東西，它都是需要有人家去做苦功的。那要去做這個苦功，我認為就代理商了。所以在現階段來看，我認為代理商它會產生一些特別的機會。相較於之前，因為我們之前對代理商的認知，就是他就是賺一個 markup 的錢。就是我今天一個產品賣進來台灣，把它倒進去你的公司裡面，我等於我是去加價賣給你。那甚至你跳過這個代理商，感覺也是可以，只是沒有辦法，你必須要透過這個代理商買。但有他沒他，好像就是說被割一手這樣而已。實際上這是很多人對代理商的一個印象，確實他們就是在做這樣的事情，就是在 markup。

可是呢，就像我們可能在兩三年前，你如果去跟人家講說，我今天要去說它廣達或是緯創的話，大家會覺得很嘲諷，就是沒有人會買這個東西。因為 ODM 跟 OEM 已經發展到一個很成熟的階段，他們做的東西什麼，可能就是一些筆電，然後跟一些傳統的一次性伺服器，所以就已經沒有什麼新鮮事了。那代理商本來在做的事情，就是純粹有去把東西倒進來，可是他們現在可能會因為某一個軟體的贏家——但我們現在還不知道這些軟體的贏家是誰——但就像是 Confluent，你在台灣是找得到他代理商是誰的，但我們不知道 Confluent 是不是贏家嘛。可是這些代理商因為他直接第一線接觸到客戶，那如果這些客戶選擇要去採用他們的服務的話，就是這些美國軟體的服務的話，他就必須要透過這個代理商，可能派他們訓練過的 team 下來幫你做。那你覺得他收的錢會是像之前這樣子，單純一個 markup 的錢？不止，他有一些服務費用跟後來的養護費用。

而且它有一件有趣的事情是，當我們今天去拆解某一個 solution，像我在網上看到一個文章還蠻有趣的，他就提到說，你今天如果說把 GPT 放進去航空公司的對話機器人的話，其實老實講，他就是一個比較厲害的搜尋引擎而已。就是他可以告訴你說，可能我們公司的機票時間怎麼樣，那現在還有哪些票種、有什麼加值服務跟行李打包。可是這東西其實就是你上網找也找得到。但是你要讓這個機器人從工人智慧變成真正的人工智慧，他必須第一個，他可以採集消費者的各樣資訊，我要可以去採集你的資訊。那再來呢，我要可以去運用我們公司跨部門所有的資源。

這東西其實他本質上需要的是，除了 GPT 之外，還有 GPT 再串接的一些軟體的服務，像是我們剛剛提到的 Apache Kafka，他把資訊流的服務把它串進去之後，我才可以去整合公司所有的資源嘛。可是當我去整合公司所有的資源的時候，我馬上去產生另外一個問題，那資安怎麼辦？那資安一樣就是透過代理商，他一定可以去找到另外一個可能專門去解決 AI-related 資安相關的服務，再把它加進去。所以某程度上是那種疊床架屋的感覺，但是就是會 work。我幫你把各種東西整合進來，但是就會 work。那我就可以賣你一個除了這些東西本來的費用，我 markup 之外，那我還幫你去把它整合起來。所以這種苦功就會變成好像有一種特別的機會存在。就有點像說，你知道其實 Dell 的伺服器，他就是跑去跟 LSI 買一個 RAID 卡，類似這樣，然後他把它轉賣，就這樣，但是我可以把你要的功能兜起來給你，我就可以賺到這樣的一筆錢。

但有些地方，就算我們可能發現代理商在後面有一些特別的商業機會，但其實代理商又有分很多種。像有些代理商他就是國內網，像國內網的話，我之前有跟股友討論，我們覺得國內網就會是一波流。也就是當大家可能看懂这个機會，並且真的有某一個軟體的推出，然後讓大家可以直接算出說他因為這個軟體可以賺多少錢，他會一波直接反應完。為什麼？因為我可以直接算出說你到底會賺多少錢，因為你的最終市場就在台灣，所以就是這樣一波就結束了。但有些人他的生意市場可能不只是在台灣，他也有做到海外去的，這個想像空間就大了。所以這是第一個我們可能會去做篩選的標準：你是國內網，還是你可以打國際盃的？不管是在代理商、SI、支付或是軟體都是這樣子。
