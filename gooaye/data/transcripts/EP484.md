最近跟朋友在討論下一個可能會出現的狀況，就是我們俗稱的「灰犀牛」，也就是大家可能已經看到它出現，已經在角落或是房間裡面，只是你不確定這個東西的破壞到底有多大。我們的做法是，等到這些東西開始有些傷害的時候，我們就會立刻去做一些反應。其實周邊都會發現市場的很多事情，都是等到發生之後再去做處理。除非有一些事情已經很確定、可以掌握週期的東西，你會提早去做佈局去蹲，那不然大多數的東西其實就是發生之後再立刻做處理其實就夠了，它會遠遠的好過你完全不會做處理。

所以不管是你可能發現一個題材，然後等到它已經開始有突破表態了才上去；或者說注意到一些黑天鵝，然後開始出現跌破的時候你再開始去做減碼，我是說各種部位控管其實都是 OK 的。那只有時候假設我們更有自信的話，我們就提早去做佈局。

接下來聊到的東西，我覺得都會比較偏向是我們有注意到它，但是我們會等到它正式發酵的時候再去做一個處理。

在美國這邊，我覺得有兩件事情。一個就是降息，如果說降得太慢的話，其實大家會擔心，因為我們已經有看到一些數據顯示，在美國這邊其實有蠻多的企業開始破產了，他們有 file for bankruptcy，那這個數字是持續在增加，雖然還沒有到那種很緊張、增加很快的一個階段，但是有持續在增加。所以確實，這個高利率是會讓一些企業去承受蠻大的一個壓力，而且已經扛很久。那現在開始降息，甚至會擔心說會不會降得不夠快，會出問題。但是如果降得太快，又可能擔心通膨會二次復發，所以就會變成說這個東西是蠻棘手、蠻難處理的。那我們只能夠等到可能正式發生之後，再來思考是往哪一個方向的可能性比較高，但是這是一個要注意的東西。

第二個就是 Kamala Harris 假如正式上任的話，這對我們來講，也是一個對資本市場可能會有不小影響的東西。雖然我們回到我們的結論，就是說應該還是會期待美國資本市場會長期向上，可是短期的一些動盪應該都會發生，而且這個動盪應該不用等到她正式上任，只要她贏面越來越大，那可能就會出現這樣的一個問題。那只是現在看起來就還好，還沒有到很嚴重的一個階段，也沒有說她一定會上，因為川普最近也開始重新拿回一些優勢，所以我們還要再觀察下去。

那 Kamala Harris 我們之前有跟大家聊說，她針對一些價格的控管，這個是我們會站在一個反對的角度，但是也不得不說短期內一定會有效果，因為他是直接類似說「我把一個價格的上限卡在那邊」，所以價格上不去，短期內看起來是好的。可是長期會出問題的原因是，當你設定一個價格的上限，美國是一個資本主義的國家，大家開始發現說「我做這個東西就是會賠錢，那我幹嘛還要做？」，我做了就是會被你價格控管，我就賠錢，我就不要做。所以因為他不做，反而攻擊的缺乏會更嚴重，那導致價格反而是漲更多。所以理論上是為了去設定一個價格上限來控制價格，但實際上造成的影響反而是會導致這些生產方他沒有誘因，所以他不去做生產，那你反而是變成價格會越來越高。那是一個會有反效果的一個做法，而且這個做法其實在過去已經經歷過多次的驗證，然後發現它是沒有用的，但她一直提出這樣的政策，所以就發現說還蠻幽默的。

第二項政策是資本市場的「未實現資本利得稅」。那也是因為這些稅務相關的東西，導致像前面有跟大家聊到一些富豪或一些投資英雄，他們有在處分他的股票，說不定就是有可能有聞到這樣的味道，所以可能提早去做一些準備，也是有機會的。因為他們在圈子裡面，應該都是會有一些管道可以得知後面的政策可能會怎麼樣規劃，所以他們可能就提早去做一些避險。

那這個東西會有一點問題的原因在於說，我覺得現有美國的稅制是 OK 的，也就是當你今天賣出一個資產、賣你的股票的時候，你股票現價有多少，跟你當時取得的成本，或是說公司配給你的成本，然後去算中間的價差，這樣去課稅，其實這樣子是還算合理的。那你把它變成是未實現資本利得稅，馬上會產生幾個問題。

第一個問題就是，那假設我的未實現資本「利損」呢？它不是利得，是利損，那你是不是要退稅給我？再來就是，你要怎麼樣去採計這個時間點？是要用可能每天的收盤價去平均嗎？還是說你是設定某一個特定的時間點？那你設定某個特定的時間點，那是不是公司也可以針對這樣去做它的股價？就是它有很多選擇的 bug 的問題。然後再來就是，其實有蠻多的 founder，特別是在草創階段，他根本就是沒有太多的現金在手上，他的身家完全是來自於他的公司，那他如果沒有賣掉的話，等於說他手上就是沒有任何現金，他假設真的要去繳稅，因為要被課未實現資本利得稅，那搞不好就會變成在公司草創階段，他就必須要去喪失他的控制權。因為國家跟他講說你要繳稅，那可能就會導致說，有些創辦人會想：「那我不要在美國上市，我給它去別的地方上。」有一些人才可能會因此流失掉。然後再來，在資本市場上面，一定也會有更多有的沒的操作。然後在政府的效率上面，就是你要去設定一個新的部門來針對這些人課稅，也是很奇怪，就是你現有的體制就好好的，為什麼要去改東西？東西沒有壞掉就不要亂修。

可是我覺得這個政策很容易推起來的主因是，絕大多數人就不是資本家，他們也沒有資產，所以他們就覺得很爽，反正不是我的事情，你去課有錢人的稅多爽。但這個最後面我覺得都會影響到大家，只是用大家可能想像不到的方式。所以這個可能也是一個對資本市場會造成動盪的事情。

那再來拉回到台灣，台灣我們覺得比較值得擔心的就是房地產，因為房地產在前陣子是真的狂噴。那雖然台灣的房地產有一個不死神話，我覺得過去可能就在什麼九六飛彈危機、SARS、然後在金融海嘯，然後可能 2014 的時候有一個緩跌之外，好像看起來就是一個只會漲的市場。但這個市場可能最近在新青安的幫助之下，熱度是真的到了一個居高不下的地方。

我們認為後面應該會有一些影響，雖然我們沒有辦法很明確的去量化，因為我們自己專注的東西是在股票市場。雖然我自己本身有房子，但是我其實不太在意我房子的市價是漲是跌，因為我雖然之前跟大家分享說，房地產資金假設說有需要的話，你是可以增貸，但我自己實際上是沒有去增貸它。那我的房地產噴了 60%，實際上我也都沒有去動它，我也沒有要賣它。所以對我來講，其實房地產的變化是形同紙上富貴，我在我自己的淨值變化上，也從來都不會去加計房地產。

但可能對一些人來講，就要說它是使用房地產去抵押做貸款，然後拿出來投資股票的，這些人可能就會在後面假設新青安造成一些影響的話，他接連一起受到影響，那可能就會導致在股票市場的資金也會受到影響。再來就是股票市場裡面，跟我們叫直接相關的一些代銷或是建商的股票，我們可能有去觀察這個趨勢，去做對應的一些操作。那只是現在都還在一個觀察的狀態。

我只是稍微跟大家分享一下，根據最新永慶房屋的一個資料統計，8 月全台房市的交易量是月減 23%，年減是 8%。那在台北是月減 18%；新竹跟新北是 26%；桃園、高雄月減 28%；台中月減 22%；台南月減 15%。然後在台灣房屋這邊，我們也是注意到類似的一個數據。也就是說，這一波的新青安熱，你要知道它其實是在差不多 8 月下旬的時候，才開始有很多的新聞報導，但這個數字是在講整個 8 月，所以實際上影響可能要再看下個月跟下下個月，我認為那個影響的幅度應該是會持續地去放大。

雖然你在新聞上看到很多說法，講說其實央行也沒有這麼狠，其實他們還在討論一些做法。可是在第一線的成交跟交屋市場，我們已經有注意到很多的狀況，不然說也有一些消息開始傳出來，但我們這邊就不去講說是哪一些建商或是哪一些建案，但是有那一種可能 80 戶、100 戶，那我整包一起成交的，然後他講說我會去幫你弄貸款，那這些人是直接一起都沒有辦法弄到貸款，那屋主欲哭無淚。就是它發生在很多地方，大家都有類似的狀況。那也有一些我們自己的聽眾，可能在討論區跟大家討論，然後也有講到自己有遇到類似的一些情形。

那現在的解方就是說，盡量去找看看還有哪些銀行有額度給它，那最終大家的名單可能就是什麼土地銀行或是農會，因為大家都知道說自己還有一些額度。但如果說大家也往這邊擠，後面這邊額度很快也被大家用光，就不知道。那現在結論就是，我認為說政府會去針對這個東西做處理，它應該是不會直接就讓房地產熄火，因為上車的人太多了，所以它會做的東西可能就是會去把一些條件弄得更加的嚴格，但是應該是不會直接讓大家死透。就是有些人會期待說直接一波把大家打死，我覺得應該是不太會發生這樣的狀況。

可能就是變成比較尷尬的局面，就是這些人可能既然買房子的時候，蠻多人他不是只是想要自住，他的想法可能是說，用買來之後可能幾年內轉手的話，我在這個寬限期內我就不用去付本金，那這樣子壓力超低。然後寬限期到，房價又漲了，實際上他可能不一定有這個能力去買這個房子，說不定那個頭期款蠻多是家人幫他出，或者是他自己本來打的如意算盤是想說，這個五年之後薪水可能更高，或是五年之後我搞不好就成家立業，搞不好有一個人可以來跟我一起分擔房貸。大家不知道，搞不好就會發現說，這陣子有蠻多的結婚潮會出現，因為大家都要找一個人幫忙分擔房貸。

反正就是各種算盤都覺得說，反正時間到了就把它賣掉就好，那不會賠到錢。那只是如果時間到了，你賣不掉怎麼辦？所以一下很多人買的物件是那一種可能比較高單價，然後低總價的，就是什麼小三房、兩房，覺得這種東西流動性很好跑。嘛，只是每個人都跟你想一樣的話，會發生什麼事情呢？會不會可能根本這五年還沒有到，就一堆東西要拿到市場上賣，然後變成完全的買方市場？更加的相反，其實我們認為說這樣的劇本發生機率是不低啦，然後是真的有可能往這個方向去發生。

那對資本市場的影響，因為我們不知道到底有多少人是使用房地產的槓桿來弄股票，所以這個還沒有辦法去算。但是我們知道的是，就是有蠻多的營建股，他們現在其實市值都幾百億，那個都很難想像，你知道他那股票漲個 10%、20%，他不知道要再多賣多少個房子，所以真的能夠一直這樣下去嗎？這也是大家的一個問號。所以我們只能夠後面再持續的觀察。但總之這就是我們有理出說，在美國有一些特定的狀況，在台灣有一些特定的狀況，這些東西都是擺在那邊，我就直接告訴你說這可能會有問題的。那只是它什麼時候會有問題，那如果你的時間點抓不好的話，其實也沒有用啦。

就是跟大家講這個並不是說什麼，所以這個東西是空下去，什麼美國賀錦麗相關的股票空下去，台灣房市空下去，不是這樣，你時間點抓錯是沒有鳥用的啦。像是我們可能在快一年前就跟大家講說 Intel 可能會有個 spin-off，最近新聞都在吵說他要 spin-off，然後什麼昨天噴個幾趴，大家說這個是因為在討論他要 spin-off。可是這個其實一些消息靈通的人是更早就知道這件事情了，但是你那時候去買股票，你到現在都是賠錢。所以時間點沒有抓好，也沒有幹用。但我們還是會去試著去做一些嘗試，如果說有一些東西明顯已經出現一些跡象的話。那只是這個後面再看，像我自己本身都還是一個超級大多頭，就是去針對多頭的東西做佈局，然後後面有變化的時候再來做一些討論。

我自己本身在林口，因為我跟大家分享說我的房子是比較大坪數的，那其實老實講在我當時買的時候，我心中想的一個想法是，這個東西會套到死，但是也沒差，因為我就是自己要住的，所以我去買一個我知道他是高總價、然後可能後面成交量也會超低的東西，因為大坪數的東西在市場上其實是不好跑的。然後我很意外的是，就是我現在有注意到說，我們社區有成交到一坪快要 50 的，所以也就是說基本上我買了大概三年左右吧，然後這個房價的單坪是翻了快要一倍，這個是我很難以想像的事情。就是我躺在家裡什麼都沒做，我就只是躺在家裡，然後就有這樣的一個變化。

對啦，其實很多指數仔也這樣子，他可能也是沒有在研究股票，就買指數。所以我們要去參與資本市場真的非常重要。只是當一個市場可能已經到一個瘋狂狀態的時候，有時候要特別小心。當然我自己是不會去把我的房子賣掉，沒有要幹嘛，就是他之後怎麼漲怎麼跌都不干我的事。那只是同時就會產生一些憂國憂民的想法，覺得說因為我家可以直接看到艾司摩爾的工地，就是在林口的工一那邊，他們現在在蓋工地，正在挖地基跟地下室。然後我心中就在想說，看這些人，他們都還不知道他們自己要來林口，就是艾司摩爾那邊可能有幾千、一萬人吧，然後旁邊還是這麼多空地，那些都是之後會進來的廠房。然後這些人他們都還不知道自己要來林口，因為他蓋好可能是 2026、2027 或什麼的。那等到他真的要來林口的時候，不知道他第一印象是覺得說林口就是一個適合成家的地方，不是蛋白嗎？不是很便宜嗎？然後一來發現說，一坪要 80 萬、90 萬，搞不好甚至要上百。你說不可能嗎？我覺得也很難講，我真的很難想像，我買的時候一坪 28，然後現在跟我講說什麼林口已經有成交到一坪 80，這到底是怎麼回事？80 在我以前認知，這你要進去板橋了吧，你已經可以買一些台北市的，你跑來住林口幹嘛？
