那在接下來我們聊一下 Intel 在電話會議裡面講的一些內容。那上次我們在節目中有跟大家聊到說，在 Lunar Lake 上面，他們是走一個 Memory on Package 的模式。那到時候會聊這個東西，是因為我們本身有在關注一些 DDR5 模組廠相關的生意的機會，像是 CKD 晶片，這個可能是在後面一個，雖然他 TAM 並不是太大，不過可能毛利然後跟獲利貢獻都不錯的一個小晶片，這個晶片是可能在市場上會產生一些機會。那以及 DDR5 本身的滲透率，如果說拉高的話，其實也會產生很多有趣的空間。所以其實當時我們回去追蹤這樣的一個題材。

那只是呢，因為 Lunar Lake 的一個評價是在市場上、在一些評測上面看起來大好，我們說像是「極客灣」直接把它吹到天上去。那所以當時是有去做一個測算，就是如果說這個模式是 OK 的話，會不會之後 Intel 就會去持續推出更多這樣子的一個產品？那繼續推出這樣子更多產品的一個問題就在於說，因為他是直接把 Memory 封進去他的晶片上面，所以本身我們去看的一些 PMIC，或是說所謂什麼 CKD 晶片，那些他的供應商，就高機率不會是臺廠。我們看起來有驗進去的 PMIC 應該是 Renesas，是 Renesas 的東西，所以臺商要打進這一塊應該是比較困難。也就是說我們的生意可能比較聚焦在模組的部分，那如果說你是直接把它封進去晶片裡面做晶片級封裝的話，那可能就不會是我們的一個生意。所以當時是有去做這樣子的一個計算。

那好在呢，在這次的電話會議裡面，Intel 算是直接表明了，Lunar Lake 就是一個 one-off，就是一次性的產品。那之後的 Panther Lake 跟 Nova Lake 可能應該都不會去走這一種把 Memory 去封進去整個晶片的一個做法。

雖然這個有一點開倒車的感覺，怎麼說？雖然理論上對沒錯，你要這樣封進去，是要做一個輕薄筆電嘛。但我想像是，你輕薄筆電應該要持續推下去，因為現在 AI 筆電的時代，你不要講說是 AI 筆電，你其實你看到的 data center 也知道，他們都盡量去把記憶體做離運算單元很近，像是 HBM 為什麼這樣做？最大的原因是因為，你記憶體靠的越近，你那個效率是越好的，延遲是越低的，表現會更好。所以大家都要往這個趨向去做，然後你媽的 Intel 竟然決定要開倒車。就是你終於做出一個，看起來在評測上面大家很喜歡的東西，那結果你竟然在電話會議裡面很驕傲的跟大家宣佈說：「大家不要擔心，因為這個 Lunar Lake 我們是走 Memory on Package，所以它的那個毛利可能比較低一點，那我們之後 Panther Lake 跟 Nova Lake 就直接把這個數字拉回來。」

看起來好像是給大家一個好訊息，但同時也給大家一個壞訊息，就是奇怪了，你一個看起來成功的賭注，為什麼你不要繼續做下去？那這邊大家有可能有幾個理由，我自己講一下我的猜想，有兩個。

一個是，你知道就 OEM、ODM 的老大哥，他們其實在筆電端這邊，他們是有一些話語權的。所以有可能說，因為我本來有一個很大的 markup 跟去增加公司整體獲利的空間，就是我可以去採購這些記憶體。那現在把記憶體拿去你自己要去做採購的話，那是不是我在 markup 變少了？那是不是以後配合度就低一點之類的？就是有可能是有收到 OEM 或是 ODM 這邊的一些評價。

那另外一個，可能比較偏向他自身本身的財務問題，就是因為他如果要把這些 Memory 直接封進去的話，他會需要去採購 DRAM。那這些 DRAM 的價格其實是還蠻貴的，LPDDR。那有可能就是說，因為他不想去扛，就是他自己的電話會議裡面講到，Lunar Lake 本身他規劃是一個 niche market，小眾的東西，那意外的搭到 AI PC，變成是一個其實是有放量的東西，放量到會影響到公司的毛利率。因為這個 Memory 直接封進去，外加外包去給別人代工，所以毛利是比較低一點的。那這樣的一個產品呢，就是因為他需要去採購大量的記憶體，所以有可能會去遭受到記憶體……你知道那個價格波動是很大的，那如果說那個價格掉下去的話，那公司可能就會非常腦闆，手上有一堆庫存會出問題。

所以應該是這兩個原因，我覺得八九不離十，而不是傻到很白目的，這個產品做得好好，大家也喜歡，我就是故意不要做。我覺得一定沒有這麼白目，應該是比較偏向，然後財務或者說在供應鏈上面有受到上下游的一些壓力，所以最後面就決定說，那可能這個產品呢，我們就只做一代。

雖然這樣子看起來一個保守的決定，好像聽起來合情合理，但我不知道，我就覺得說有時候你該賭的時候，你要衝一發。就是這個東西是可行的，你應該是要繼續的去推它，而不是說出一代，然後後面我就不要做，我要去回到白牌的那個模式。雖然白牌那個模式，對我們在追供應鏈來講是比較好一點，因為臺系供應鏈在那樣的一個模式比較有賺錢的機會。但是因為我自己本身也是有持有一些 Intel，我覺得說奇怪，一個看起來會 work 的東西，因為大家很喜歡，你為什麼不要繼續推？那個牙膏終於擠出來了，然後要把牙膏收回去，就你到底在幹嘛？

但看起來公司派很高興：「這東西很讚、很棒、大家都喜歡，但重點是我們有賺錢，大家最重要的，你不要擔心，我這個毛利數字會回去。」大家應該不想要看這個短期的毛利數字，而是到底有沒有成功回去推出一些產品線，然後是真正的受到大家喜歡，然後重新的衝返榮光，而不是說短期的這個數字要維持住。其實這個也是我覺得蠻多公司會掉入一個陷阱，就是它過度的去追求一些短線上的數字，因為有一些投資人就是隻要看這些數字。可是你為了要去滿足這些投資人的想法，或是說老闆追求自己的所謂 KPI，然後去滿足這些短線數字，你公司會忽略掉一些長線的佈局。所以有些看起來現在會痛的東西，你說毛利差一點或什麼，你就是應該要繼續做下去。

所以我又覺得，比較可能性的，就是因為它可能去扛這個 LPDDR5 的庫存，它會覺得壓力太大，所以它才決定不要這樣做。因為他自己去採購，那如果我現在是把他下放給 OEM、ODM 的話，他們也高興，對我來講，我壓力也沒有這麼大。應該是這個機率比較大一點，不然沒有道理，一個看起來是很成功的產品，終於要重返榮光，然後供應鏈卻說要喊停。不知道，還是說他們其實覺得，就算記憶體沒有靠這麼近，然後在後面也不會有顯著的差距？我不知道，這個可能我們要在後面再來觀察看看。但我確實相信，對大多數的消費者來講，就是那些很追求登峰造極的，你說什麼延遲要再少個一些什麼的，那個其實真的大多數人感覺不出來。所以說不定他們真的在下一個很大的一盤棋，他知道說，在這樣的話之下，我們財務是 OK，那我的供應鏈也是 OK，然後外加其實消費者感應也不會太明顯，然後他就做出這樣的決定。說不定之後看是一個 OK 的決定。

所以我們再稍微觀察一下，但現階段我們可以稍微的忽略，就是在過去的，忘記是一集還兩集，那時候我聊到，就是說他去走 Lunar Lake 這樣的一個模式，那並且放量，然後並且可能其他的廠商之外，都全部往這個方向推的話，你要小心一些臺系供應鏈，他可能吃不到這個單。這樣的一個說法，就可以直接把它畫掉，因為沒事了，Intel 這邊已經宣佈說這個東西我只要做一代。所以稍微跟大家更新到這邊。
