## 戒菸啟示：你愛的可能不是尼古丁，是深呼吸

我昨天看到一個非常有趣的影片分享，突然間就意會到為什麼以前會這麼喜歡抽菸。以前我自己很喜歡抽菸，可能早上一根，或是你要拉屎拉不出來的時候抽一根，這個有抽菸的應該知道，整個就順暢了，覺得抽菸真的好像是讓我們的身體可以按下開機鍵的感覺。

但其實某個程度上有點像是說先幫你弱化，然後弱化之後可能讓你補充尼古丁，你就再一次的強化回去。所以其實嚴格上來說，有點像是回到本來的一個狀態，可是那樣的一個落差就會讓你覺得很爽。我現在抽菸是抽一個寂寞，抽一個自以為自己是詩人，很有感覺，或是覺得自己帥或什麼。那時候的感受是我真的非常喜歡抽菸，我也不知道為什麼我非常喜歡抽菸，但我只要抽菸我心情就很好。

後來看了這個影片之後，我就可以明白為什麼自己會喜歡抽菸。他就說，其實你要想，把抽菸這件事情本身拉掉，它是一件很好的事情。什麼意思呢？你什麼時候會抽菸？早上醒來的時候走到陽台跑去抽菸；可能去參加一個派對，在裡面跟大家待在那種二氧化碳濃度超高的地方，整個都已經不太舒服了，跑去外面呼吸一下，那時候我們都想出去呼吸一下；或是吃熱炒吃到很飽，突然覺得需要幫助消化一下，就走到外面去抽菸；或是今天可能在一個非常繁忙的會議，接著覺得需要讓自己放鬆一下，就走出去抽菸。

仔細想想這些時刻，如果說你可以走到室外去抽菸，有抽菸的朋友應該會覺得這就是最爽的時間點，或是說完事之後、上廁所之前之類的。但仔細想想，這些時候其實本身就是很適合去深呼吸的時候。所以或許你喜歡的，部分是菸跟尼古丁，部分是深呼吸這件事情。如果說你可以在繁忙的會議之後走到戶外去深呼吸一下，吃完熱炒超飽的時候走到戶外去深呼吸一下，在一個非常繁忙的派對裡面，你已經覺得自己快要 wear out、那個刺激太強了，走到外面只有你一個人在那邊呼吸一下，或者說跟你的好朋友走到室外吸一口氣，你就會發現其實你喜歡的不一定全然是尼古丁跟菸，而是那個呼吸的感覺。

所以去抽菸這個行為，把抽菸拉掉就是去深呼吸——一個人去深呼吸、在壓力之後去深呼吸，或是在早上起來之後走到陽台去深呼吸。我突然想通這件事情之後，過去幾天就試看看，就過往（雖然我已經有點忘記了，因為已經是好幾年前）自己會想到抽菸的時刻，就走到外面去深呼吸一下，突然發現真的會覺得腦袋變得很清楚，覺得自己的狀態變得很好。所以或許大家可以往這個方向去切入看看。

那時候看到真的覺得太有趣了，如果有人早一點跟我講的話，那時候就不用瘋狂吃零食、每天咬口香糖、貼尼古清貼片，還有那種口福的、用吸的、吸起來會辣辣的，讓你腦袋會有一點 boost 的感覺，所以你就不會想一直去抽菸。那時候是用了各種方法，但我現在意外的發現，其實你只要把菸放下，走到外面，還是可以拿著你的零卡可樂，搭配呼吸的話，其實是還蠻不錯的。

這個事情之所以這麼讓人成癮，很有可能是因為它本身結合了我們人類最原始的一個需求——你需要在很繁忙、壓力很大的時候，人太多刺激過強的時候，需要一個人，或是跟自己的三五好朋友走到外面，大家都不說話，稍微的深呼吸一下。那一兩分鐘的深呼吸真的會改變很多事情。所以這個一定要跟大家分享一下，因為我覺得怎麼會沒有人在當時跟我說這個東西，或是我怎麼沒有看到這樣的一個說法，非常有意思。

總之分享是希望大家可以有效的去戒菸，其實菸跟酒都是比較不好的東西，能夠少一點就少一點。酒的話我覺得難免，有時候可能身邊特別是年紀比較大的長輩朋友，大家出來的時候可能還是會喝一些。當然我自己本身是沒有飲酒的習慣，就不會像自己家裡的一些長輩，每天中午就一定要喝一點，沒有喝絕對生氣，很奇怪，我想說幹差小，這你也很奇怪。

可能酒就是在一些聚會喝，但我自己有稍微注意到——可能因為我朋友都是年紀比我大蠻多的，比較少同輩或比我年輕的朋友——可是有時候跟自己的同輩或比較年輕的朋友出來就會注意到，其實大家喝的酒越來越少。甚至像新一代年輕人，2000 年以後的那些人，他們蠻多是完全不喝酒的。甚至他們有些 party 是辦在麵包店裡面，大家喝汽水或喝茶之類的，沒有在喝酒精，他們不需要靠酒精就可以很酷。

不像我們可能老一輩的，大家就會覺得有時候你就是喝酒之後才放鬆啊，才可以去談事情啊，或是才可以去表述你的真感情啊，那代表你平常很ㄍㄧㄥ嘛、很假嘛。可是現在新一代年輕人沒有在跟你五四三的，你進去一個工作，像以前怎麼樣，你當學徒今天學三年——你現在學什麼？慘犯慘犯慘了一年之後，再教你什麼烤海苔之類的，他一進去你不教他的話，他就不學啦。除非你真的是急屌店，不然他就不要了，他去別家店就好。

所以我覺得新一代年輕人其實才是真正的活出自我，他們不會去受到各種的限制，不會去受到什麼工作的話術，或是社交的習慣跟禮儀。怎麼大家就是要請客菸？你現在有人會帶請客菸也傻小？再來就是有些環境一定要喝酒、一定要準備什麼？現在新一代人真的沒有在跟你搞這種東西。

所以某程度上我也算是去跟年輕人接軌，雖然嚴格上來說我應該不算老人吧，30 幾歲，但就是慢慢的向他們看齊。很多時候我看新一輩的年輕人會覺得這才是真正生活的態度。特別是在現在的所謂 AI 時代之下，你發現像是生產力本身是越來越廉價的東西。以前的碼農可能是很貴的職業，現在不一定要用到這麼貴的碼農，搭配一些 token fee 的話，你可以產出很棒的效果。所以甚至大家在討論說未來有一天會不會有 universal basic income 的導入，因為我們不再需要這麼多的生產力。所以這些人其實就是提早做好這樣的準備，他們不會對工作上面有一些執念。像以前我們如果老闆跟我講「我分你股份」或「我讓你當店長」，我可能就拿命去拼；現在你要分他股份，他還是會先問你「那我可以準時下班嗎？不行那我不要」。但其實這可能才是一個正確的生活態度。

所以年輕人看起來是不要有太多這種成癮的項目，或是一些對身體不好的東西要稍微注意一下。這個其實也是說給我自己聽，因為偶爾還是會不小心放縱一下，可是我現在已經是非常有意識的在節制。也希望大家可以在這些成癮物的東西上面去做好一定程度的控制。但人生可能還是有一些成癮的東西，像我電玩成癮，我就會說如果沒有電玩成癮的話，很多時候我也不知道我在生命之中還可以幹嘛。所以有些你喜歡的東西、你喜歡去吃的美食、打電動，或是有時候你還是喜歡去抽個菸喝個酒，可能都還可以，只是要繼續節制，不要讓它變成是你的 boss、你的 type way，叫你做什麼就要做什麼，那就比較不酷。稍微跟大家分享一下自己的心得。

## 市場觀察：CPU 族群走出來與多頭輪動

接下來我們來進入市場的話題。最近大家有看到一些關於中東的各種酒的廟的東西 [聽不清楚]，你就會注意到，就像我們上集跟大家講的，一般來說你看到這些大 K 出來之後，後面的一些消息就會比較鈍化一點。當然現在很明顯是又再次拉到一些壓力的位置，台股這邊已經拉到新高非常屌，甚至世界屌，所以已經是多頭沒有什麼任何的疑問。

美股來到一些壓力的位置，雖然我們會直接把站到季線上面就視為現在又再次回到多頭行情，但後面可能有一些消息再出來的話，有機會就會再往下打。如果下面都撐得住的話，那應該就沒有問題，有可能這一波的修正就會把它當成是結束了。

同時呢，在過去一個禮拜我自己算是過得非常開心，就是可能終於在美股我今年第二季的支出終於有賺到錢的味道。第一季的時候只有少數的時間待在水上，大多數時間都在水下。最主要的問題就是，當然用事後論據看都非常簡單——早知道就全部買光通就好，可是手上不是全部都光通嗎，你有晶片、你有軟體、你有一些有的沒有的東西，你有巨頭，他們可能在過去的表現就是沒有這麼好。所以第一季的時候，即便手上有光通的部位噴得非常凶，可是因為你有太多的拖油瓶，所以導致整體的表現並不是這麼好。

但是過去一陣子已經順利的回正，而且是快速的往上爬。爬的速度是我自己都有嚇到。有時候就是說，選擇會比努力重要。你在那些很垃圾的標的上努力了半天，然後發現就算可能抓了很好的買賣點，可是還是無法避免的看著你的鏡子（部位）不停的往下掉，因為我們不會去做 all in all out，你只能夠做一些調節。所以簡單來講，巧婦難為無米之炊，這個東西就在往下掉，你再怎麼會調也都沒有用。但現在在資金的規劃之下，把錢丟到像是 CPU，或者說 Marvell，或是像最近的高速傳輸其實也重新復活，光通又再次創新高，所以已經非常有效的在資金的調動之下，持續的去攻新高了。希望今年可以在美股也做出非常好的表現。

過去數集我們都一直跟大家聊到 CPU 相關的議題。一般在我們自己的做法就是，我們會有一個想法，這個想法我們會用真金白銀到市場裡面去做驗證——這就是大家最怕的，反正每次有什麼想法就是一定會拿錢去做實驗——如果對的話就會持續去做加碼。所以如果真的有咬到東西的話，其實都會在往上爬的這個階段爬得非常快。也希望後面這個 CPU 可以持續有更多發揮的空間。

最近 NVIDIA 也去入股了一家叫 SiFive，是一個在做 RISC-V 的公司。這個我算是有聽到身邊產業圈的朋友跟我聊到，他說其實現在大家把目光放在 CPU 這邊，因為知道在 Agentic AI 時代，很多工作除了我們知道 GPU 算力之外，必須要靠 CPU 就像是教練一樣去做調度。大量的用到 CPU，那當然 RISC-V 這樣的架構也是有可能去做出特化版的 CPU，所以它有可能會重新被市場關注。

再來就是在 Custom Made 的 ASIC，最近也開始獲得很多關注。像 Amazon 這邊就有提到，有客戶想要跟他們全包下 Graviton 的一個 CPU 的算力，但是他們直接跟他說不好意思我們自己都不夠用。你要知道 AWS 是一個在 CSP 裡面的 powerplay，他們有非常大的容量，所以 Andy Jassy 講了這句話，其實隱含的是——我相信在未來的一小段時間（當然我們還不確定這個趨勢會不會形成），大家應該可以稍微注意一下 AWS 的 Supply Chain，特別是可能跟 Graviton 有關的。因為之前可能大家會看的都是比較偏向是 GPU，或是加速運算的 ASIC，但他現在就是在提到 CPU 這邊也是非常的缺乏，有兩家客戶去問可不可以直接跟你全包，他直接跟他說我們自己要用。

這什麼意思呢？這個意思就是說，要嘛亞馬遜會在後面加大 CPU 的各種投片跟生產，去供這些客戶——這個是我覺得就必然會發生的事情。第二就是這些客戶因為他們要 CPU，可是他們沒有 CPU，所以他必須要去找別人去拿 CPU，或甚至是去找一些比較偏向像 GPGPU 這樣的廠商，他們是做通用品的，就 Intel、AMD 之類的，他們就是有機會在後面都會產出不錯的成績。在自股東信裡面已經像是洩題了，告訴你 CPU 非常好。

所以在我自己的角度會認為都持續的看下去，手上的 CPU 股票就是持續拿好。甚至會開始去思考有沒有可能走出，除了這個 2B 的很好之外，會影響到 2C。類似可能我們在之前的一些電話會議裡面已經有看到，像 Intel 就有提到，他們是為了要去做這些 2B 的東西，會去排擠到 2C 的東西，甚至是刻意為之。反而 2C 也可能會因為這樣產出一些機會，就是在大家都一面倒不看好，認為今年來看手機或 PC 都會很差的時候，有沒有可能在這邊也會另外產出機會。但我覺得最保守的做法當然還是先鎖定 server 端的 CPU，或是 custom made 的 CPU，如果有一些額外機會產生的話，當然可以往消費端的角度看過去。所以這是在未來都必須要非常認真去注意的項目。

終於有咬到東西的感覺，非常的爽快。過去三個月能夠講的就只有廣東（光通），因為光通是美股裡面少數表現很好。可是現在 CPU 也走出來，所以美股你已經開始注意到有一點不一樣了，已經有一些新的族群走出來了。這其實非常重要——一個多頭行情就是要很多頭，多頭就是很多頭，就是一堆東西都輪流在那邊向前衝。所以像台股就是非常經典的一個示範，有各種類股在那邊輪動，這就是一個很健康的多頭。所以美股現在除了光通之外，開始有一些新的族群跑出來，那是非常好。我們就持續觀察美股有沒有可能藉由族群的輪動，然後也跑去攻新高，這是非常重要的事情。

## Anthropic Methos 與 AI Doomsday 行銷

再來我們稍微聊一下 Claude 這邊，就是 Anthropic 他們推出的 Methos。這個 Methos 並沒有對外開放，他只先開放給一些策略夥伴，他們的用詞就是說因為這東西太強大了，所以我們不能夠公佈。老實講這個我有一點想要吐槽。

但我們知道，以目前的模型來講，我覺得 Anthropic 就是無庸置疑是最猛的，特別是在 2B 的應用，它是 dominant 的地位。當然其實這些 AI 的模型 provider，他們很多時候都會不停的去——我覺得是這樣子，因為他們有很龐大的壓力，就像他們的老闆自己也曾經提到，如果我的成長性慢下來，我會直接破產，因為我花太多錢去投入在算力裡面。所以他們必須要維持一個非常漂亮的成長率，為了這個成長率，其實我覺得有時候他們會講出很聳動的話。

像之前 Sam Altman，他一開始出來講的時候，我們會去認真聽他講話，可是後來大家應該有注意到，我的節目就有點轉向，就是我不太喜歡這個人，我覺得他講話有點過度的尬吹了。現在我慢慢覺得 Anthropic 也是有一樣的味道出來，整天就是「啊我們的東西太危險了」、「我們的東西可能會影響到什麼」、「美國政府可能要跟我簽什麼才可以使用我的東西」。但我知道你的東西真的好用、真的強，因為我們自己也是第一線有在使用 AI 產品的人。

大家講說那個尬吹的成分就高到——像他們對外公布的一個信條說，我們懷疑我們的 AI 自己有一些想法，因為他可能會自己去生成一些故事性內容的東西。不知道，他就叫做 LLM，他本來的工作就是生成這些東西。可是我覺得他們有點 delusional，他們就說這個就代表他要產生一些智慧或什麼的。聽到這邊一些新的聽眾可能就會覺得這個傢伙是 AI 黑嗎？不是，其實我們是 AI 狂粉，但很多時候我們會想要去踩一些煞車，因為我覺得真的推過頭。

特別是每次 Anthropic 出來推東西的時候，就會導致市場裡面的腥風血雨。市場裡面的泛軟體股，過去其實有一些比較強勢的重新爬回去，像 Cloudflare 或是 Palantir，或是 CrowdStrike，他們都曾經爬到零軸附近，等於說今年沒有跌。可是這一波他們釋出新產品，然後再一次的把這些東西再倒回去打一次，那我就覺得蠻靠北的，因為我自己也是長期持有這些標的。所以就會覺得市場的這些 FUD 情緒已經讓我自己覺得有點煩躁了。

如果說他講說他的 AI 東西有多屌、多會偵測什麼的，那他們之前怎麼會發生自己的代碼外洩？你 AI 真的這麼厲害，那應該會把所有洞都封起來。或者說在 Reddit 上面也有看到，有些人去看了 Anthropic 他們宣稱的做法之後，他們用更便宜的模型其實一樣可以找出那些所謂的漏洞之類的。所以嚴格上來說，當然 AI 工具在各個工作環節上不可否認的都一定有幫助到我們，可是 Anthropic 這家公司有把這件事情放到很大，變成有點像是在尬吹的狀態。

他們尬吹的方式都是這樣——「我覺得有東西可能真的太危險會影響到大家」。可是我認為任何明眼人都知道，之所以他現在沒有辦法把那個 Methos 放出來的原因是因為他的算力不夠。簡單來講就是他的算力不夠，這個東西放出來的話，大家一定會把它罵爆。為什麼？因為其實現在大家已經知道，像你去使用 Anthropic 的東西，在某些時段——特別是美國人起床的時段，你會注意到你的 AI 變笨了。為什麼？因為太多人去做使用，他們的算力是不夠。

這也是過去我們提到，像 Google 或其他的 AI 競爭者，時不時會出現的問題，就是當太多人去使用它，它一開始很強，可是當太多人去使用就會變笨。所以如果今天它推出 Methos，用到更多的算力，一定會讓大家感覺到其實這個東西並沒有像它講的這麼聰明，它只有在少部分的人去做使用的時候，可以是比較強的狀態。所以這東西其實也不是只有 Methos，包含它前面的一些產品也是一樣，如果今天用的人沒有這麼多，其實會變得是比較聰明一點。

至於什麼時候可以解決這樣的問題呢？很多 AI 產品包含像 Apple、OpenAI、Google，他們推出來一開始都超強，可是為什麼後面大家都會開始去靠北說要變笨？因為算力是供不應求的狀態，所以算力供應的失衡必須要快速去解決，才有辦法解決這件事情。可是目前是沒有辦法快速解決，特別是當大家發現 GPU 或 Accelerator 的 ASIC 目前已經是 OK、相對的問題了，可是又再一次跑出來 CPU，這個是很難以去估計到底需求有多少的東西。

所以嚴格上來說這些軍火業者，像 TSM 或是後端的封裝、M-Core、SPIL 或是相關的先進製程要用到的各種東西，應該都還可以再好過一陣子。因為以目前來看就是，大家本來講說 GPU 或 ASIC，可是現在你還要再多看一個 CPU。等到 CPU 可能都完美的去做了解決之後，說不定又有新的 AI 模型要用到更多的元件，所以目前看起來這個需求還是有點像無底洞的狀態。

我認為其實像 Methos 沒有辦法對外推出，並不是像他們所提的——當然他提到那可能也是一環，這個拿出來可能會被人家拿去攻擊或 blah blah blah——可是像這樣去尬吹自己東西有多厲害，這不是只有台灣這樣做，美國也會這樣做。你其實去看 2019 年的時候，OpenAI 就有出來講說他的東西有多厲害、會有多恐怖之類，可是已經 2026 年的我們，世界還是沒有毀滅。所以像過去大家講什麼天網要出來了、Terminator 什麼 AI 要讓世界毀滅，最近發現 AI 其實真的不會讓世界毀滅，會讓世界毀滅都是人類，人類去做白癡的事情，不用 AI 也是可以讓世界毀滅。

接下來我們關注的重點就是，等他這個東西推出來，可能就代表他已經 secure 到夠多的算力，他就會把它放出來，並不是大家講的這個東西真的有多恐怖、多嚇人。因為其實過去每個 AI 廠商都想要這樣去跟大家講，他有一點行銷的術語在裡面。可能就等到算力去做解決之後，他可以把這個東西端出來，不然端出來很高程度的會——怎麼講——你把那個預期打得太滿了。

所以很多時候大家會注意到，每次有一個新的模型出來，前兩天你去滑社群都是尬吹文，怎麼太屌了、那個比分又怎麼樣、用起來多強，然後大概兩個禮拜之後就開始一堆人來罵。所以也會是類似的狀態，只要他沒有把算力解決，都會是這樣。

至於那些可能殺下去的軟體股會怎麼樣，我個人是認為那算是我的長期持股，所以像我自己會繼續抱著，可能等到下一波他們開始要進攻的時候，我們才會再去額外做注意。不然這陣子我覺得自己也是蠻意外，像 Anthropic 這樣的一個 AI Doomsday Code，每次都把這些 SaaS 或是相關的軟體應用跟整合商打下去，我以為早就該告一個段落了，但還是持續在發生中。所以目前很明顯資金沒有在這個地方，可能要等到後續再來觀察什麼時候會讓市場稍微冷靜一下，然後接著再找一些介入的機會。

## 被動元件缺貨與光通代工新趨勢

最近在台股也有蠻有趣的事情，應該說兩件事。首先快速聊一下被動元件，被動元件我們算是可能在兩三個月前就持續有去注意，可是兩三個月前我自己覺得我那時候買太早，所以我有認輸。認輸之後後來又有再回頭繼續做介入，現在就在思考要不要把這個部位放到很大。下個禮拜會有一個比較重大的事件，禮拜三國巨要開法說會，沒有意外的話，法說會的內容應該是會全面性的正面。

為什麼？因為我們現在已經注意到，很多被動元件都開始出現 D time 是往上拉的狀態。它還沒有像記憶體這樣全面的缺貨，可是已經在像是 Tantalum Cap（鉭電容）或是 SP CAP、鋁電容上面已經注意到嚴重的缺貨。MLCC 已經在緊接線附近，MLCC 目前其實在市場裡面最缺的應該是會比較偏向像是村田，或是像是 Samsung 所做出來的比較高階 AI 應用的。

當然這個比較有點類似 2017 的狀態，那時候也是可能在一支手機或是一個電動車裡面，要用到超多的被動元件，導致一些高階廠商去往高階的方向走，一些比較低階的產品就放出來。它其實有點類似的邏輯——有一個主軸產品，以前可能是手機或電動車，現在就是 AI 伺服器，所以有機會就是會變成一波新的缺貨潮。

我自己也有在臉書分享這個文章，很多時候我們都拋出影域，故意丟出來看大家會怎麼講。有一些產業的朋友就有來給我 feedback，基本上產業的朋友都有看到這樣的趨勢，只是他們的說法是這有點類似 23 或 24 的記憶體，就是你不知道什麼時候它會發作。那時候發作的一個原因是因為 Sora，因為 Sora 就整個觸動到整個記憶體族群噴出去，還有那時候有看到 SanDisk 的漲價信，我們在第一時間就有跟大家分享。

所以可能就是，當然漲價信我們過去在被動元件已經看到很多了，所以它可能需要某一個事件去做催化跟驅動。如果有那個事件跑出來的話，可能資金就會往這個方向去看。接下來可能注意一下禮拜三國巨的法說會講什麼東西。沒有任何意外的話，我們已經看到他下個 Q 也是要持續的漲價。再來就是他旗下的 Kemet（基美），也是持續在談電容上面去做漲價。所以被動零件非常的缺，這應該是沒有什麼太多的爭議，到後面還會再更缺。

接下來另外一個有趣的題材也在這邊跟大家分享一下。總之就是現在有蠻多的光通廠，不管是美系的（其實有些是美系跟台股，身體裡面是台股的），再來或是台灣自己的一些光通相關元件廠，我們最近有注意到一個趨勢。相信這個股票圈的人士應該大家都有發現——前面大家買的線爐是怎樣？線爐就是去幫人家做代工 shuffle box。後來大家跑去買了一個網通廠，為什麼？因為那個網通廠有 SMT 產線。然後在過去一陣子，大家跑去買一些 LED 的打件廠，為什麼跑去買那個 LED 打件廠？因為就有在講說美系的這些客人們，他們現在需要這些產線。

要怎樣？他可能就是自己去自帶這些設備進來，那設備應該也是甲方去付的——可能這些巨頭，他們需要這些光通相關的元件，所以他就付錢給這個廠商買了設備之後，然後這個廠商可能再去找另外一個有廠地、有產線的廠商，把我的設備放進去做生產。像這樣子的標的在台股是越來越多，現在已經是上升到大概有 4 檔是這樣。所以我開始不免認為這會不會有機會變成是某一種新的趨勢。

首先大家先知道 SMT 打件廠，今天講說把一些電子元件弄到板子上面，其實毛利本身並不是一個太高的東西。可是過去可能有蠻多的閒置——好像有些在做這個 LED 的，他那個 SMT 的打件其實是非常的完善，只是說那個產線可能是沒有辦法把它放滿，所以在過去一陣子大家都不會喜歡它，誰會喜歡這種沒有在題材上的東西。可是近期發生的事情是因為，可能光通的需求真的太大了，所以他們必須要把很多產能生出來。自己可能去做擴廠，又需要去過環評啊，土地取得不易啊，要買廠房可能也沒這麼簡單。

那有一個可能相對簡單的做法怎麼樣？我直接去找既有的廠商，他即便跟我的本業根本就是沒有關係，反正沒有關係，你就是給我你的場地跟人力就好，那可能設備或是相關的 know how，我自己會進去幫你去做處理。像這樣子的標的在台股算是越來越多了，所以這有可能是接下來的新趨勢。

但是我自己稍微跟朋友討論了一下，這個你很難提早去布局——就覺得說你現在可不可以去找一些有 SMT 產能、然後你知道它的產線稼動率並沒有很高，你就先去買，然後問會不會有客人來找他？這個可能在投報上面就沒有這麼好。所以我們的做法是把這些股票全部撈出來，然後接著去做一些調查。但如果我們沒有辦法調查的話是最好的——如果沒有辦法調查到誰轉強，就是有可能有一些新的故事在背後，只是這個可能被我們排在介入順序的 tier 會比較後面的東西，因為它有點像是在賭博。

可是我們自己已經有看到的廠商是有 4 家，他們的狀態就是反正因為光通這些客人他們的需求太大，所以他們就把這個社會背後是有的沒有的東西然後委託你在你這邊去做生產。分到的錢雖然不多，可是對這些廠商來講，他可能現在是一個虧損的狀態，所以不無小補，毛利即便是可能就 10% 上下，或甚至更低，可是就是賺錢的東西，而且他可能來的量是很大的。所以這可能也是在後面可以稍微注意的一些新話題。

我相信後面市場就會有一些什麼新聞那些東西討論，因為這個太誇張了，我們也沒有想過怎麼會這樣，怎麼會這麼多人會來找這些八竿子打不著的。當然我會講說光通某個程度上也是一種泛網通的題材，可是一個是做可能不知道那種 CP 小東西的，你怎麼可能會去做光通的東西？後來才知道是因為——簡單講——東方有貴人來了，貴人要他們去拓產、先弄東西、有什麼的，那就導致原來現在可以有這麼多的訂單就這樣憑空生出來。

這個可能是說你投資這家公司的時候，你根本就不知道。像我那時候去看到我們提到這幾家公司，我就故意會去滑一下討論區，看大家怎麼看，然後大家還在那邊瞎掰理由，他們不知道實際上是有這個來自美國或是台灣的一些大客戶看上你的產線，所以是有這個背後的故事在這邊。當然這個其實我覺得他可以視為是一個可能深水區的東西，一般人比較不易輕易的去介入，因為他可以把東西給你，當然他也可以把東西抽走，或者說他可以說要給你，但是他可能還在驗證的階段，最後沒有給你，可是股價是炒到恨天高，後來就會崩下來。所以還是要小心，但我們還是會把自己有注意到的一些機會都跟大家分享跟討論一下。
